4.10 Riemann integration \((\star )\)

In this section, our aim is to show that if a bounded function \(f: [a,b] \rightarrow \R \) is Riemann integrable, then it is measurable and Lebesgue integrable. Moreover, in this case the Riemann and Lebesgue integrals of \(f\) are equal. We state this result formally as Theorem 4.10.3. In this section we will prefer to say â€˜\(f\) is Lebesgue integralâ€™ rather than \(f\in \Lone \), simply because it makes for better grammar when we compare Riemann and Lebesgue integrability within the same sentence.

We begin by briefly revising the Riemann integral. Note that this whole section is marked with a \((\star )\), meaning that it is off-syllabus. It will be discussed briefly in lectures.

4.10.1 The Riemann integral \((\star )\)

A partition \(\cal P\) of \([a, b]\) is a set of points \(\{x_{0}, x_{1}, \ldots , x_{n}\}\) with \(a = x_{0} < x_{1} < \cdots < x_{n-1} < x_{n} = b\). Define \(m_{j} = \inf _{x_{j-1} \leq x \leq x_{j}}f(x)\) and \(M_{j} = \sup _{x_{j-1} \leq x \leq x_{j}}f(x)\). We underestimate by defining

\[ L(f, {\cal P}) = \sum _{j=1}^{n}m_{j}(x_{j} - x_{j-1}),\]

and overestimate by defining

\[ U(f, {\cal P}) = \sum _{j=1}^{n}M_{j}(x_{j} - x_{j-1}),\]

A partition \({\cal P}^{\prime }\) is said to be a refinement of \(\cal P\) if \({\cal P} \subset {\cal P}^{\prime }\). We then have

\(\seteqnumber{0}{4.}{20}\)

\begin{equation} \label {ref} L(f, {\cal P}) \leq L(f, {\cal P}^{\prime }),~~~ U(f, {\cal P}^{\prime }) \leq U(f, {\cal P}). \end{equation}

A sequence of partitions \(({\cal P}_{n})\) is said to be increasing if \({\cal P}_{n+1}\) is a refinement of \({\cal P}_{n}\) for all \(\nN \).

Now define the lower integral \(L_{a,b}f = \sup _{{\cal P}}L(f, {\cal P})\), and the upper integral \(U_{a,b}f = \inf _{{\cal P}}U(f, {\cal P})\). We say that \(f\) is Riemann integrable over \([a, b]\) if \(L_{a,b}f = U_{a,b}f\), and we then write the common value as \(\int _{a}^{b}f(x)dx\). In particular, every continuous function on \([a, b]\) is Riemann integrable. The next result is very useful:

  • Theorem 4.10.1 The bounded function \(f\) is Riemann integrable on \([a, b]\) if and only if for every \(\eps > 0\) there exists a partition \(\cal P\) for which

    \(\seteqnumber{0}{4.}{21}\)

    \begin{equation} \label {part} U(f, {\cal P}) - L(f, {\cal P}) < \eps . \end{equation}

If (4.22) holds for some \(\cal P\), it also holds for all refinements of \(\cal P\). A useful corollary is:

  • Corollary 4.10.2 If the bounded function \(f\) is Riemann integrable on \([a, b]\), then there exists an increasing sequence \(({\cal P}_{n})\) of partitions of \([a,b]\) for which

    \[ \lim _{n \rightarrow \infty }U(f, {\cal P}_{n}) = \lim _{n \rightarrow \infty }L(f, {\cal P}_{n}) = \int _{a}^{b}f(x)dx\]

Proof: This follows from Theorem (4.10.1) by successively choosing \(\eps = 1, \frac {1}{2}, \frac {1}{3}, \ldots , \frac {1}{n}, \ldots .\) If the sequence \(({\cal P}_{n})\) is not increasing, then just replace \({\cal P}_{n}\) with \({\cal P}_{n} \cup {\cal P}_{n-1}\) and observe that this can only improve the inequality (4.22).   ∎

We can integrate many more functions using Lebesgue integration than we could using Riemann integration. For example, with Riemann integration we could not conclude that \(\int _{[a,b]}{\1}_{\R \sc \mathbb {Q}}(x) dx = (b - a)\), but with Lebesgue integration we can.

4.10.2 The connection \((\star )\)

  • Theorem 4.10.3 If \(f: [a, b] \rightarrow \R \) is Riemann integrable, then it is Lebesgue integrable, and the two integrals coincide.

Proof: We use the notation \(\lambda \) for Lebsgue measure in this section. We also write \(M = \sup _{x \in [a,b]}|f(x)|\) and \(m = \inf _{x \in [a,b]}|f(x)|\).

Let \(\cal P\) be a partition as above and define simple functions,

\[ g_{\cal P} = \sum _{j=1}^{n}m_{j}{\1}_{(x_{j-1}, x_{j}]}, ~~~h_{\cal P} = \sum _{j=1}^{n}M_{j}{\1}_{(x_{j-1}, x_{j}]}.\]

Consider the sequences \((g_{n})\) and \((h_{n})\) which correspond to the partitions of Corollary 4.10.2 and note that

\[ L_{n}(f) = \int _{[a,b]}g_{n}d\lambda ,~~~U_{n}f = \int _{[a,b]}h_{n}d\lambda ,\]

where \(U_{n}(f) = U(f, {\cal P}_{n})\) and \(L_{n}(f) = L(f, {\cal P}_{n})\). Clearly we also have for each \(\nN \),

\(\seteqnumber{0}{4.}{22}\)

\begin{equation} \label {order} g_{n} \leq f \leq h_{n}. \end{equation}

Since \((g_{n})\) is increasing (by (4.21)) and bounded above by \(M\), it converges pointwise to a measurable function \(g\). Similarly \((h_{n})\) is decreasing and bounded below by \(m\), so it converges pointwise to a measurable function \(h\). By (4.23) we have

\(\seteqnumber{0}{4.}{23}\)

\begin{equation} \label {order1} g \leq f \leq h. \end{equation}

Again since \(\max _{n \in \N }\{|g_{n}|, |h_{n}|\} \leq M\), we can use dominated convergence to deduce that \(g\) and \(h\) are both integrable on \([a,b]\) and by Corollary 4.10.2,

\[ \int _{[a, b]}g d \lambda = \lim _{n \rightarrow \infty }L_{n}(f) = \int _{a}^{b}f(x)dx = \lim _{n \rightarrow \infty }U_{n}(f) = \int _{[a, b]}h d \lambda .\]

Hence we have

\[ \int _{[a, b]} (h - g)d \lambda = 0,\]

and so by Corollary 3.3.1, \(h(x) = g(x)\) (a.e.). Then by (4.24) \(f = g\) (a.e.) and so \(f\) is measurable3 and also integrable. So \(\int _{[a, b]}f d \lambda = \int _{[a, b]}g d\lambda \), and hence we have

\[ \int _{[a, b]}f d \lambda = \int _{a}^{b}f(x)dx.\]

  ∎

3 Iâ€™m glossing over a subtlety here. It is not true in general, that a function that is almost everywhere equal to a measurable function is measurable. It works in this case due to a special property of the Borel \(\sigma \)-field known as â€˜completenessâ€™.

4.10.3 Discussion \((\star )\)

An important caveat is that Theorem 4.10.3 only applies to bounded closed intervals. On unbounded intervals, there are examples of functions are Riemann integrable4 but not Lebesgue integrable. One such example is \(\int _0^\infty \frac {\sin x}{x}\,dx\). The function \(\frac {\sin x}{x}\) oscillates above and below \(0\) as \(x\to \infty \), and the Riemann integral \(\int _0^\infty \frac {\sin x}{x}\,dx=\lim \limits _{X\to \infty }\int _0^X \frac {\sin x}{x}\,dx\) only exists because these oscillations cancel each other out. In Lebesgue integration this isnâ€™t allowed to happen, and \(\frac {\sin x}{x}\) fails to be Lebesgue integrable because \(\int _0^\infty |\frac {\sin x}{x}|\,dx=\infty \). In fact, \(\int _0^\infty (\frac {\sin x}{x})_-\,dx=\int _0^\infty (\frac {\sin x}{x})_+\,dx=\infty \), so \(\int _0^\infty |\frac {\sin x}{x}|\,dx\) is undefined for the Lebesgue integral.

Letâ€™s discuss these ideas in the context of infinite series which, as we showed in Example 4.7.5, are a special case of the Lebesgue integral. That is,

\[\int _\N a_n\,d\#(n)=\sum _{n=1}^\infty a_n\]

where \(a:\N \to \R \) is a sequence, and \(\#\) is the counting measure on \(\N \). Note that \((a_n)\) is integrable if and only if \(\sum _n|a_n|<\infty \), which is usually referred to as â€˜absolute convergenceâ€™ in the context of infinite series. The key is that when infinite series are absolutely convergent they are much better behaved, as the following result shows. A â€˜re-orderingâ€™ of a series simply means arranging its terms in a different order.

4 Strictly, we should say â€˜improperlyâ€™ Riemann integrable.

  • Theorem 4.10.4 Let \((a_n)\) be a real sequence.

    • 1. Suppose \(\sum _{n=1}^\infty |a_n|=\infty \) and \(a_n\to 0\). Then, for any \(\alpha \in \R \), there is a re-ordering \(b_n=a_{p(n)}\) such that \(\sum _{i=1}^n b_n\to \alpha \).

    • 2. Suppose \(\sum _n|a_n|<\infty \). Then, for any re-ordering \(b_n=a_{p(n)}\), we have \(\sum _{n=1}^\infty a_n=\sum _n b_n\in \R \).

Imagine if we allowed something similar to case 1 was allowed to happen in integration, and let us think about integration over \(\R \). It would mean that re-ordering the \(x\)-axis values (e.g. swap \([0,1)\) with \([1,2)\) and so on) could change the value of \(\int _\R f(x)\,dx\)! This would be nonsensical, and mean that integration over \(\R \) no longer had anything to do with â€˜area under the curveâ€™. So we have to avoid it, and we do so by restricting to integrable functions. Only then can we find nice conditions for â€˜limitâ€™ theorems like the dominated convergence theorem. Lebesgue integration solves this problem; Riemann integration cannot.
