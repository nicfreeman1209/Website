4.2 The Cumulative Distribution Function

Let \(X:\Omega \rightarrow \R \) be a random variable. Its cumulative distribution function or cdf is the mapping \(F_{X}:\R \rightarrow [0, 1]\) defined for each \(x \in \R \) by

\[F_{X}(x) = \P [X \leq x] = p_{X}((-\infty , x]).\]

When \(X\) is clear from the context we might write \(F\) instead of \(F_X\).

The next result gathers together some useful properties of the cdf. Recall that if \(f:\R \rightarrow \R ,\) the left limit at \(x\) is \(\lim _{y \uparrow x}f(y) = \lim _{y \rightarrow x, y < x}f(y)\), and the right limit at \(x\) is \(\lim _{y \downarrow x}f(y) = \lim _{y \rightarrow x, y > x}f(y)\). In general, left and right limits may not exist, but they do if the function \(f\) is monotonic increasing (or decreasing).

  • Theorem 4.2.1 Let \(X\) be a random variable having cdf \(F\).

    • 1. \(\P [X > x] = 1 - F(x)\),

    • 2. \(\P [x < X \leq y] = F(y) - F(x)\) for all \(x < y\).

    • 3. \(F\) is monotone increasing, i.e \(F(x) \leq F(y)\) for all \(x < y\),

    • 4. \(\P [X=x] = F(x) - \lim _{y \uparrow x}F(y)\),

    • 5. The mapping \(x \rightarrow F(x)\) is right continuous, i.e. \(F(x) = \lim _{y \downarrow x}F(y)\), for all \(x \in \R \),

    • 6. \(\lim \limits _{x \rightarrow -\infty }F(x) = 0\) and \(\lim \limits _{x \rightarrow \infty }F(x) = 1.\)

Proof: Parts 1-3 are left as (easy!) exercises for you. Part 6 is Problem 4.2.

Part 4: Let \((a_{n})\) be a sequence of positive numbers that decreases to zero. Let \(x \in \R \) be arbitrary and for each \(\nN \), define \(B_{n} = (x - a_{n} < X \leq x)\). Then \((B_{n})\) decreases to the event \((X = x)\), so using part 2 and Theorem 4.1.1,

\[\P [X = x] = \lim _{n \rightarrow \infty }\P [B_{n}] = F(x) - \lim _{n \rightarrow \infty }F(x - a_{n}),\]

and the result follows.

Part 5: Let \(x\) and \((a_{n})\) be as in (4) and for each \(\nN \) define \(A_{n} = (X > x + a_{n})\). The sets \((A_{n})\) are increasing to \((X > x)\), so using part 1 and Theorem 4.1.1 we find that

\[ 1 - F(x) = \lim _{n \rightarrow \infty } \P [A_{n}] = 1 - \lim _{n \rightarrow \infty }F(x + a_{n}),\]

and the result follows.   ∎

  • Remark 4.2.2 By combining parts 4 and 5 of Theorem 4.2.1, we have that \(\P [X=x_0]=0\) if and only if \(F_X(x)\) is continuous at \(x=x_0\).

  • Remark 4.2.3 (\(\star \)) It can be shown that a function \(F:\R \rightarrow \R \) is the cdf of some random variable \(X\) if and only if it satisfies properties 3, 5 and 6 of Theorem 4.2.1.
