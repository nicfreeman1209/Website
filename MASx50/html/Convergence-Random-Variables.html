5.2 Convergence of Random Variables

Let \((X_{n})\) be a sequence of random variables, all of which are defined on the same probability space \((\Omega , {\cal F}, \P )\). There are various different ways in which we can examine the convergence of this sequence to a random variable \(X\) (which is also defined on \((\Omega , {\cal F}, \P )\). They are called modes of convergence.

When we talk about convergence of real numbers \(a_n\to a\) we only have one mode of convergence, which we might think of as convergence of the value of \(a_n\) to the value of \(a\). Random variables are much more complicated objects; they take many different values with different probabilities. For this reason, there are multiple different modes of convergence of random variables.

We say that \((X_{n})\) converges to \(X\)

  • â€¢ in distribution if whenever \(\P [X=x]=0\) we have \(\P [X_n\leq x]\to \P [X\leq x]\).

  • â€¢ in probability if given any \(a > 0\), we have \(\P [|X_{n} - X| > a] \to 0\) as \(n\to \infty \),

  • â€¢ almost surely if \(\P [X_{n}(\omega ) \to X(\omega ),\text { as }n\to \infty ]=1\),

  • â€¢ in \(L^p\) if \(\E [|X_{n} - X|^{p}] \to 0\) as \(n\to \infty \).

When \((X_{n})\) converges to \(X\) almost surely we sometimes write \(X_{n} \rightarrow X\) a.s. as \(n \rightarrow \infty \). We may also write the type of convergence above the arrow e.g. \(X_n\stackrel {L^2}{\to }X\) or \(X_n\stackrel {a.s.}{\to }X\).

For \(L^p\) convergence we are usually only interested in the cases \(p=1\) and \(p=2\). The case \(p=2\) is sometimes known as convergence in mean square. For reasons that are outside of the scope of this course, \(L^p\) convergence is only defined for \(p\in [1,\infty )\).

Happily, there are some relationships between these different modes of convergence.

  • Lemma 5.2.1 Let \(X_n,X\) be random variables.

    • 1. If \(X_n\stackrel {\P }{\to } X\) then \(X_n\stackrel {d}{\to } X\).

    • 2. If \(X_n\stackrel {a.s.}{\to } X\) then \(X_n\stackrel {\P }{\to } X\).

    • 3. If \(X_n\stackrel {L^p}{\to } X\) then \(X_n\stackrel {\P }{\to } X\).

    • 4. Let \(1\leq p< q\). If \(X_n\stackrel {L^q}{\to } X\) then \(X_n\stackrel {L^p}{\to } X\).

    In all other cases (i.e. that are not automatically implied by the above), convergence in one mode does not imply convergence in another.

Proof:

  • 1. Let \(x\in \R \) be such that \(\P [X=x]=0\) and let \(\epsilon >0\). We have

    \(\seteqnumber{0}{5.}{2}\)

    \begin{align} \P [X\leq x] &=\P [X\leq x, |X_n-X|<\epsilon ]+\P [X\leq x, |X_n-X|\geq \epsilon ]\notag \\ &\leq \P [X_n\leq x+\epsilon ]+\P [|X_n-X|\geq \epsilon ].\label {eq:pd1} \end{align} Putting \(x-\epsilon \) in place of \(x\) we obtain

    \(\seteqnumber{0}{5.}{3}\)

    \begin{align} \P [X\leq x-\epsilon ] &\leq \P [X_n\leq x]+\P [|X_n-X\geq \epsilon ].\label {eq:pd2} \end{align} Combining (5.3) and (5.4) leads to

    \(\seteqnumber{0}{5.}{4}\)

    \begin{align*} \P [X\leq x-\epsilon ]-\P [|X_n-X|\geq \epsilon ] \;\leq \; \P [X_n\leq x] \;\leq \; \P [X\leq x+\epsilon ]+\P [|X_n-X|\geq \epsilon ]. \end{align*} By Remark 4.2.2 and the fact that \(\P [X=x]=0\) we have that \(y\mapsto \P [X\leq y]\) is continuous at \(y=x\). Hence, letting \(\epsilon \to 0\), both \(\P [X\leq x+\epsilon ]\) and \(\P [X\leq x-\epsilon ]\) converge to \(\P [X\leq x]\). Since limits preserve weak inequalities this gives

    \(\seteqnumber{0}{5.}{4}\)

    \begin{align*} \P [X\leq x]-\P [|X_n-X|\geq \epsilon ] \;\leq \; \P [X_n\leq x] \;\leq \; \P [X\leq x]+\P [|X_n-X|\geq \epsilon ]. \end{align*} Letting \(n\to \infty \) and using that \(X_n\stackrel {\P }{\to } X\), the sandwich rule gives \(\P [X_n\leq x]\to \P [X\leq x]\).

  • 2. Let \(\epsilon > 0\) be arbitrary and let \(A_{n} = \bigcup _{m = n}^{\infty }\{|X_{m} - X| > \epsilon \}\). Then \((A_{n})\) is a decreasing sequence of events. Let \(A = \bigcap _{n=1}^{\infty }A_{n}\). If \(\omega \in A\) then \(X_{n}(\omega )\) cannot converge to \(X(\omega )\) as \(n \rightarrow \infty \) and so \(\P [A] = 0\), because \(X_n\to X\) almost surely. By Theorem 4.1.1 part (2), \(\lim _{n \rightarrow \infty }\P [A_{n}] = \P [A] = 0\). But then by monotonicity,

    \[ \P [|X_{n} - X| > \epsilon ] \leq \P [A_{n}] \rightarrow 0~\mbox {as}~n \rightarrow \infty .\]

  • 4. \((\star )\) Weâ€™ll prove part 4 next, because it will be helpful in part 3 below. This part is non-examinable because we need an inequality that is (just) outside of our own syllabus. It is true that \(\E [|X|]^p\leq \E [|X|^p]\) for all \(p\geq 1\) and all random variables \(X\). This is a special case of a general result known as Jensenâ€™s inequality. Putting \(|X_n-X|^p\) into the inequality, and putting \(q/p\geq 1\) in place of \(p\), we have \(\E [|X_n-X|^p]^{q/p}\leq (\E [|X_n-X|^{p(q/p)}])\). Thus \(\E [|X_n-X|^p]\leq (\E [|X_n-X|^{q}])^{p/q}\). The result follows.

  • 3. Thanks to part 3, it suffices to prove that \(L^1\) convergence implies convergence in probability. From Markovâ€™s inequality (Lemma 3.3.2) for any \(a > 0\) we have

    \[\P [|X_{n} - X| > a] \leq \ds \frac {\E [|X_{n} - X|]}{a}.\]

    If \(X_n\stackrel {L^1}{\to }X\) then the right hand side tends to zero as \(n\to \infty \), hence so does the left.

  ∎

No other relationships between these modes hold in general, but if we allow extra conditions or slightly weaker conclusions then it is possible to say more. For example:

  • Theorem 5.2.2 If \(X_{n} \rightarrow X\) in probability as \(n \rightarrow \infty \) then there is a subsequence of \((X_{n})\) that converges to \(X\) almost surely.

Proof: If \((X_{n})\) converges in probability to \(X\), for all \(c > 0\), given any \(\epsilon > 0\), there exists \(N(c) \in \mathbb {N}\) so that for all \(n > N(c)\), \(\P [|X_{n} - X| > c] < \epsilon .\)

In order to find our subsequence:

â€“ First choose, \(c = 1\) and \(\epsilon = 1/2\), then for \(n > N(1)\), \(\P [|X_{n} - X| > 1] < 1/2.\)

â€“ Next choose \(c = 1/2\) and \(\epsilon = 1/4\), then for \(n > N(2)\), \(\P [|X_{n} - X| > 1/2] < 1/4.\)

â€“ In general, choose \(c = 1/r\) and \(\epsilon = 1/2^{r}\), then for \(n > N(r)\), \(\P [|X_{n} - X| > 1/r] < 1/2^{r}.\)

Now choose the numbers \(k_{r} = N(r) + 1\), for \(r \in \mathbb {N}\) to obtain a subsequence \((X_{k_{r}})\) so that for all \(r \in \mathbb {N}\),

\[ \P [|X_{k_{r}} - X| > 1/r] < 1/2^{r}.\]

Since \(\sum \frac {1}{2^{r}} < \infty \), by the first Borel-Cantelli lemma (Lemma 5.1.1) we have

\[\P [|X_{k_{r}} - X| > 1/r\;\;i.o.] = 0,\]

and so

\[\P [|X_{k_{r}} - X| \leq 1/r\;\;e.v.] = 1.\]

Hence, almost surely, there exists some \(R\in \N \) such that for all \(r\geq R\) we have \(|X_{n_{k_{r}}} - X| < 1/r\), which implies that \(|X_{n_{k_{r}}} - X|\to 0\). Hence \(X_n\to X\) almost surely.

  ∎
