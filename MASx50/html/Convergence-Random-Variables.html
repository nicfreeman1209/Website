7.2 Convergence of random variables

Let \((X_{n})\) be a sequence of random variables, all of which are defined on the same probability space \((\Omega , {\cal F}, \P )\). There are various different ways in which we can examine the convergence of this sequence to a random variable \(X\) (which is also defined on \((\Omega , {\cal F}, \P )\). They are called modes of convergence.

When we talk about convergence of real numbers \(a_n\to a\) we only have one mode of convergence, which we might think of as convergence of the value of \(a_n\) to the value of \(a\). Random variables are much more complicated objects; they take many different values with different probabilities. For this reason, there are multiple different modes of convergence of random variables.

We have already mentioned almost sure convergence of random variables in Section 5.1. We recall this and introduce three new modes of convergence here. We say that \((X_{n})\) converges to \(X\)

  • â€¢ in distribution if whenever \(\P [X=x]=0\) we have \(\P [X_n\leq x]\to \P [X\leq x]\).

  • â€¢ in probability if given any \(a > 0\), we have \(\P [|X_{n} - X| \geq a] \to 0\) as \(n\to \infty \),

  • â€¢ almost surely if \(\P [X_{n} \to X,\text { as }n\to \infty ]=1\),

  • â€¢ in \(\Lp \) if \(\E [|X_{n} - X|^{p}] \to 0\) as \(n\to \infty \).

When \((X_{n})\) converges to \(X\) almost surely we sometimes write \(X_{n} \rightarrow X\) a.s. as \(n \rightarrow \infty \). We may also write the type of convergence above the arrow e.g. \(X_n\stackrel {\Ltwo }{\to }X\) or \(X_n\stackrel {a.s.}{\to }X\).

For \(\Lp \) convergence we are usually only interested in the cases \(p=1\) and \(p=2\). The case \(p=2\) is sometimes known as convergence in mean square. For reasons that are explained in Section 4.9.2, convergence in \(\Lp \) is only defined for \(p\in [1,\infty )\).

Happily, there are some relationships between these different modes of convergence.

  • Lemma 7.2.1 Let \(X_n,X\) be random variables.

    • 1. If \(X_n\top X\) then \(X_n\tod X\).

    • 2. If \(X_n\toas X\) then \(X_n\top X\).

    • 3. If \(X_n\toLp X\) then \(X_n\top X\).

    • 4. Let \(1\leq p< q\). If \(X_n\toLq X\) then \(X_n\toLp X\).

    No other relationships exist in general, other than those implied by the above results.

Proof: The last part follows from the counterexamples in Exercises 7.3-7.5. Weâ€™ll give proofs of parts 1-4 here.

Part 1. Let \(x\in \R \) be such that \(\P [X=x]=0\) and let \(\eps >0\). We have

\(\seteqnumber{0}{7.}{2}\)

\begin{align} \P [X\leq x] &=\P [X\leq x, |X_n-X|<\eps ]+\P [X\leq x, |X_n-X|\geq \eps ]\notag \\ &\leq \P [X_n\leq x+\eps ]+\P [|X_n-X|\geq \eps ].\label {eq:pd1} \end{align} Putting \(x-\eps \) in place of \(x\) we obtain

\(\seteqnumber{0}{7.}{3}\)

\begin{align} \P [X\leq x-\eps ] &\leq \P [X_n\leq x]+\P [|X_n-X\geq \eps ].\label {eq:pd2} \end{align} Combining (7.3) and (7.4) leads to

\(\seteqnumber{0}{7.}{4}\)

\begin{align*} \P [X\leq x-\eps ]-\P [|X_n-X|\geq \eps ] \;\leq \; \P [X_n\leq x] \;\leq \; \P [X\leq x+\eps ]+\P [|X_n-X|\geq \eps ]. \end{align*} By Remark 5.2.2 and the fact that \(\P [X=x]=0\) we have that \(y\mapsto \P [X\leq y]\) is continuous at \(y=x\). Hence, letting \(\eps \to 0\), both \(\P [X\leq x+\eps ]\) and \(\P [X\leq x-\eps ]\) converge to \(\P [X\leq x]\). Since limits preserve weak inequalities this gives

\(\seteqnumber{0}{7.}{4}\)

\begin{align*} \P [X\leq x]-\P [|X_n-X|\geq \eps ] \;\leq \; \P [X_n\leq x] \;\leq \; \P [X\leq x]+\P [|X_n-X|\geq \eps ]. \end{align*} Letting \(n\to \infty \) and using that \(X_n\stackrel {\P }{\to } X\), the sandwich rule gives \(\P [X_n\leq x]\to \P [X\leq x]\).

Part 2. Let \(\eps > 0\) be arbitrary and let \(A_{n} = \bigcup _{m = n}^{\infty }\{|X_{m} - X| \geq \eps \}\). Then \((A_{n})\) is a decreasing sequence of events. Let \(A = \bigcap _{n=1}^{\infty }A_{n}\). If \(\omega \in A\) then \(X_{n}(\omega )\) cannot converge to \(X(\omega )\) as \(n \rightarrow \infty \) and so \(\P [A] = 0\), because \(X_n\to X\) almost surely. By Lemma 5.1.1 part (2), \(\lim _{n \rightarrow \infty }\P [A_{n}] = \P [A] = 0\). But then by monotonicity,

\[ \P [|X_{n} - X| \geq \eps ] \leq \P [A_{n}] \rightarrow 0~\mbox {as}~n \rightarrow \infty .\]

Part 4. \((\Delta )\) Weâ€™ll prove part 4 next, because it will be helpful in part 3 below. This part is marked with a \((\Delta )\) because we need an inequality that comes from Chapter 6. It is true that

\(\seteqnumber{0}{7.}{4}\)

\begin{equation} \label {eq:jensen_p} \E [|X|]^p\leq \E [|X|^p] \end{equation}

for all \(p\geq 1\) and all random variables \(X\). For general \(p\), this is a special case of Jensenâ€™s inequality, which is part of the independent reading in Section 6.3, marked with a \((\Delta )\). However, the \(p=2\) case is a consequence of the Cauchy-Schwartz inequality and has already appeared in Exercise 5.11. The \(p=1\) case is simply \(|\E [X]|\leq \E [|X|]\), which is the absolute value property from Theorem 4.5.3 written in the notation of probability.

Putting \(|X_n-X|^p\) into (7.5), and then putting \(q/p\geq 1\) in place of \(p\), we obtain that \(\E [|X_n-X|^p]^{q/p}\leq (\E [|X_n-X|^{p(q/p)}]).\) Thus \(\E [|X_n-X|^p]\leq (\E [|X_n-X|^{q}])^{p/q}\). The result follows.

Part 3. Thanks to part 4, it suffices to prove that \(L^1\) convergence implies convergence in probability. From Markovâ€™s inequality (Lemma 4.2.3) for any \(a > 0\) we have

\[\P [|X_{n} - X| \geq a] \leq \ds \frac {\E [|X_{n} - X|]}{a}.\]

If \(X_n\stackrel {L^1}{\to }X\) then the right hand side tends to zero as \(n\to \infty \), hence so does the left.   ∎

  • Remark 7.2.2 The following diagram records which modes of convergence imply which other modes of convergence.

    In all other cases (i.e. that are not automatically implied by the above), convergence in one mode does not imply convergence in another.

Recall that for real numbers, if \(a_n\to a\) and \(a_n\to b\) then \(a=b\), which is known as uniqueness of limits. For random variables, the situation is a little more complicated: if \(X_n\stackrel {\P }{\to } X\) and \(X_n\stackrel {\P }{\to } Y\) then \(X=Y\) almost surely. By Lemma 7.2.1, this result also applies to \(\stackrel {L^p}{\to }\) and \(\stackrel {a.s.}{\to }\). However, if we have only \(X_n\stackrel {d}{\to } X\) and \(X_n\stackrel {d}{\to } Y\) then we can only conclude that \(X\) and \(Y\) have the same distribution, that is \(\P [X\leq x]=\P [Y\leq x]\) for all \(x\). Proving these facts is Exercise 7.6.

Establishing convergence in distribution, probability and \(L^p\) usually comes down to calculating (or estimating) the important quantities involved: \(\P [X\leq x]\), \(\P [|X_n-X|\leq a]\) and \(\E [|X_n-X|^p]\), and then thinking about their limits as \(n\to \infty \). There are several examples of this type within the exercises. Almost sure convergence is harder to work with, but here we can often use the Borel-Cantelli lemmas.

  • Example 7.2.3 Let \((X_n)\) be a sequence of i.i.d. random variables, each with the uniform distribution on \([0,1]\). Then \(\P [X_n\leq \frac 13]=\P [X_n\geq \frac 23]=\frac 13\), so (using independence) by two applications of the second Borel-Cantelli lemma we have \(\P [X_n\leq \frac 13\text { i.o.}]=\P [X_n\geq \frac 23\text { i.o.}]=1\). Hence, with probability \(1\), the sequence \(X_n\) will oscillate infinitely often between \([0,\frac 13]\) and \([\frac 23,1]\), in which case it cannot converge. Thus \((X_n)\) does not converge almost surely (to any limit) in this case.

    Alternatively, consider a sequence \((Y_n)\) with \(\P [Y_n=1]=\frac {1}{n^2}\) and \(\P [Y_n=0]=1-\frac {1}{n^2}\). Since \(\sum \frac 1{n^2}<\infty \), the first Borel-Cantelli lemma tells us that \(\P [Y_n=1\text { i.o.}]=0\). Hence \(\P [Y_n=0\text { e.v.}]=1\), which implies that \(\P [Y_n\to 0]=1\), or in other words that \(Y_n\stackrel {a.s.}{\to }0\).

  • Lemma 7.2.4 If \(X_{n} \rightarrow X\) in probability as \(n \rightarrow \infty \) then there is a subsequence of \((X_{n})\) that converges to \(X\) almost surely.

Proof: If \((X_{n})\) converges in probability to \(X\), for all \(c > 0\), given any \(\eps > 0\), there exists \(N(c) \in \mathbb {N}\) so that for all \(n \geq N(c)\), \(\P [|X_{n} - X| > c] < \eps .\)

In order to find our subsequence:

â€“ First choose, \(c = 1\) and \(\eps = 1/2\), then for \(n \geq N(1)\), \(\P [|X_{n} - X| > 1] < 1/2.\)

â€“ Next choose \(c = 1/2\) and \(\eps = 1/4\), then for \(n \geq N(2)\), \(\P [|X_{n} - X| > 1/2] < 1/4.\)

â€“ In general, choose \(c = 1/r\) and \(\eps = 1/2^{r}\), then for \(n \geq N(r)\), \(\P [|X_{n} - X| > 1/r] < 1/2^{r}.\)

Set \(k_{r} = \max \{N(1),N(2),\ldots ,N(r),r\}\) for \(r \in \mathbb {N}\). to obtain a subsequence \((X_{k_{r}})\) so that for all \(r \in \mathbb {N}\),

\[ \P [|X_{k_{r}} - X| > 1/r] < 1/2^{r}.\]

Since \(\sum \frac {1}{2^{r}} < \infty \), by the first Borel-Cantelli lemma (Lemma 7.1.1) we have

\[\P [|X_{k_{r}} - X| > 1/r\;\;i.o.] = 0,\]

and so

\[\P [|X_{k_{r}} - X| \leq 1/r\;\;e.v.] = 1.\]

Hence, almost surely, there exists some \(R\in \N \) such that for all \(r\geq R\) we have \(|X_{{k_{r}}} - X| < 1/r\), which implies that \(|X_{{k_{r}}} - X|\to 0\). Hence \(X_{{k_r}}\to X\) almost surely.   ∎

  • Remark 7.2.5 A subtle point is that the subsequence of \((X_n)\) constructed by Lemma 7.2.4 is a deterministic subsequence, in the sense that \(r\mapsto k_r\) is a deterministic function. Of course, \(X_{k_r}\) is a random variable for each \(r\).
