4.7 Calculations with the Lebesgue integral

From now on, weâ€™ll often write \(\int _S f(x)\,dm(x)\) in place of \(\int _S f\,dm\). This notation has the advantage that we can write an expression like \(\int _S e^{-x^2}\,d\lambda (x)\) without having to specify the function \(f(x)=e^{-x^2}\) in advance. We follow common convention and write \(\int \ldots dx\) for integration with respect to Lebesgue measure, which we would formally write as \(\int \ldots d\lambda (x)\). Weâ€™ll allow ourselves to do this in cases where its clear from the context that we mean to integrate with respect to Lebesgue measure.

  • Remark 4.7.1 \((\star )\) In some countries (e.g. France) it is common to write \(\int _S dm\,f\) instead of \(\int _S f\,dm\). We wonâ€™t use that notation within this course. The notation \(\int _S f(x) m(dx)\) in place of \(\int _S f(x) dm(x)\) is also widely used.

We will show in Theorem 4.10.3 that, if \(f:[a,b]\to \R \) is Riemann integrable, over a closed bounded interval \([a,b]\sw \R \), then \(f\) is also Lebesgue integrable, and in this case the value of the two integrals is equal. Consequently, you may use all the facts you already know about Riemann integration on \(\R \) to evaluate integrals of the form \(\int _{[a,b]} f(x)\,dx\), which we would normally write as \(\int _a^b f(x)\,dx\). This includes integration by substitution, by parts, the Fundamental Theorem of Calculus, and so on. A word of warning is necessary: such results only apply on bounded intervals \([a,b]\) and in general are not true on unbounded intervals.

The Lebesgue integral also gives us the MCT and DCT, for when we need limits and integrals to interact. In this section we look at a few examples of using the major results of Chapter 4 to integrate particular functions. Weâ€™ll begin with the testing for \(\Lone \) and using the monotone convergence theorem, before moving on to the dominated convergence theorem.

  • Example 4.7.2 We aim show that \(f(x) = x^{-\alpha }\) is in \(\Lone \) on \([1, \infty )\) for \(\alpha > 1\).

    For each \(\nN \) define \(f_{n}(x) = x^{-\alpha }{\1}_{[1,n]}(x)\). Then \((f_{n}(x))\) increases to \(f(x)\) as \(n \rightarrow \infty \). We have

    \[\int _{1}^{\infty }f_{n}(x)dx = \int _{1}^{n}x^{-\alpha }dx = \frac {1}{\alpha - 1}(1 - n^{1 - \alpha }).\]

    By the monotone convergence theorem,

    \[ \int _{1}^{\infty }x^{-\alpha }dx = \frac {1}{\alpha - 1}\lim _{n \rightarrow \infty }(1 - n^{1 - \alpha }) = \frac {1}{\alpha - 1}.\]

  • Example 4.7.3 We aim to show that \(f(x) = x^{\alpha }e^{-x}\) is in \(\Lone \) on \([0, \infty )\) for \(\alpha > 0\).

    The key idea here is that \(e^{-x}\) tends to zero very quickly as \(x\to \infty \), and we can use this fast convergence to overpower the â€˜opposingâ€™ fact that \(x^\alpha \to \infty \). Recall that for any \(M \geq 0\) we have \(\lim _{x \rightarrow \infty }x^{M}e^{-x} = 0\), so that given any \(\eps > 0\) there exists \(R > 0\) so that \(x > R \Rightarrow x^{M}e^{-x} < \eps \), and choose \(M\) so that \(M - \alpha > 1\). Now write

    \[ x^{\alpha }e^{-x} = x^{\alpha }e^{-x}{\1}_{[0,R]}(x) + x^{\alpha }e^{-x}{\1}_{(R,\infty )}(x).\]

    By part (b) of Exercise 4.4 the sum of two \(\Lone \) functions is in \(\Lone \), so weâ€™ll aim to prove that both terms on the right hand side are in \(\Lone \). The first term on the right clearly is, because it is bounded on \([0,R]\) and zero elsewhere. For the second term we use that fact that for all \(x > R\),

    \[ x^{\alpha }e^{-x} = x^{M}e^{-x}.x^{\alpha - M} < \eps x^{\alpha - M},\]

    and which is thus in \(\Lone \) by Example 4.7.2. So the result follows by Lemma 4.6.1.

  • Example 4.7.4 We want to calculate

    \[\lim _{n\to \infty } \int _0^1 \frac {nx^{2}}{nx + 5}\,dx.\]

    Weâ€™ll work in the measure space \(([0,1], {\cal B}([0, 1]), \lambda )\) and consider the sequence of functions \((f_{n})\) where \(f_{n}(x) = \frac {nx^{2}}{nx + 5}\) for all \(x \in [0,1], \nN \). Each \(f_{n}\) is continuous, hence also measurable by Lemma 3.2.1. It is straightforward to check that \(\limn f_{n}(x) = x\) for all \(x \in [0,1]\) and that \(|f_{n}(x)| \leq 1\) for all \(\nN , x \in [0,1]\). So in this case, we can take \(K = 1\), and apply the DCT with dominating function \(g=1\) to deduce that \(f(x) = x\) is in \(\Lone \), and

    \[ \limn \int _{[0,1]}\frac {nx^{2}}{nx + 5}\,dx = \int _{[0,1]}x \,dx.\]

    We finish by evaluating \(\int _{[0,1]}x\, dx = \left [\frac {x^{2}}{2}\right ]_{0}^{1} = \frac {1}{2}.\)

  • Example 4.7.5 Summation of series is a special case of Lebesgue integration. Suppose that we are interested in \(\sum _{n=1}^{\infty }a_{n}\), where \(a_{n} \geq 0\) for all \(\nN \). We consider the sequence \((a_{n})\) as a function \(a:{\mathbb N} \rightarrow [0, \infty )\). We work with the measure space \(({\mathbb N}, {\mathcal P}({\mathbb N}), m)\) where \(m\) is counting measure. Then every sequence \((a_{n})\) gives rise to a non-negative measurable function \(a\) and

    \[ \sum _{n=1}^{\infty }a_{n} = \int _{{\mathbb N}}a(n)\,dm(n),\]

    which is for you to show in Problem 4.10. The same formula holds for general \((a_n)\sw \R \) provided that \(\sum _n |a_n|<\infty \) i.e. that \(a\in \Lone (\N )\).

    In this context the monotone and dominated converge theorems provide tools for working with sequences of series. For example, suppose that for each \(m\in \N \) we have a sequence \((a_m(n))_{n\in \N }\), given by

    \[a_m(n)=\frac {1}{n^3}+\frac {1}{1+m^2n^2}.\]

    We canâ€™t easily compute the value of \(\sum _{n\in \N } a_m(n)\) for any given \(m\). But we can note that \(a_m(n)\to \frac {1}{n^3}\) as \(m\to \infty \), for all \(n\), and that \(|a_m(n)|\leq g(n)=\frac {1}{n^3}+\frac {1}{n^2}\). We know from analysis that \(\sum _n \frac {1}{n^3}\) and \(\sum _n \frac {1}{n^2}\) are both finite, so \(g\in \Lone (\N ,\mc {P}(\N ),m)\) â€“ which in this setting is just the claim that \(\sum _n g(n)<\infty \). So the Dominated Convergence Theorem applies, and we obtain

    \[\lim _{m\to \infty }\sum _{n\in \N }\l (\frac {1}{n^3}+\frac {1}{1+m^2n^2}\r )=\sum _{n\in \N }\frac {1}{n^3}.\]
