3.8 Calculations with the Lebesgue Integral

Now lets look at a few examples of using the major results of Chapter 3 to integrate particular functions. Weâ€™ll begin with the testing for integrability and using the monotone convergence theorem, before moving on to the bounded and dominated convergence theorems.

From now on, weâ€™ll often adopt a piece of common notation and write \(\int \ldots dx\) for integration with respect to Lebesgue measure, which formally we would write as \(\int \ldots \lambda (dx)\). Weâ€™ll allow ourselves to do this in cases where its clear from the context that we mean to integrate with respect to Lebesgue measure.

  • Example 3.8.1 We aim show that \(f(x) = x^{-\alpha }\) is integrable on \([1, \infty )\) for \(\alpha > 1\).

    For each \(\nN \) define \(f_{n}(x) = x^{-\alpha }{\1}_{[1,n]}(x)\). Then \((f_{n}(x))\) increases to \(f(x)\) as \(n \rightarrow \infty \). We have

    \[\int _{1}^{\infty }f_{n}(x)dx = \int _{1}^{n}x^{-\alpha }dx = \frac {1}{\alpha - 1}(1 - n^{1 - \alpha }).\]

    By the monotone convergence theorem,

    \[ \int _{1}^{\infty }x^{-\alpha }dx = \frac {1}{\alpha - 1}\lim _{n \rightarrow \infty }(1 - n^{1 - \alpha }) = \frac {1}{\alpha - 1}.\]

  • Example 3.8.2 We aim to show that \(f(x) = x^{\alpha }e^{-x}\) is integrable on \([0, \infty )\) for \(\alpha > 0\).

    The key idea here is that \(e^{-x}\) tends to zero very quickly as \(x\to \infty \), and we can use this fast convergence to overpower the â€˜opposingâ€™ fact that \(x^\alpha \to \infty \). Recall that for any \(M \geq 0\) we have \(\lim _{x \rightarrow \infty }x^{M}e^{-x} = 0\), so that given any \(\epsilon > 0\) there exists \(R > 0\) so that \(x > R \Rightarrow x^{M}e^{-x} < \epsilon \), and choose \(M\) so that \(M - \alpha > 1\). Now write

    \[ x^{\alpha }e^{-x} = x^{\alpha }e^{-x}{\1}_{[0,R]}(x) + x^{\alpha }e^{-x}{\1}_{(R,\infty )}(x).\]

    By part 2 of Theorem 3.6.2 here, the sum of two integrable functions is an integrable function, so weâ€™ll aim to prove that both terms on the right hand side are integrable. The first term on the right clearly is, because it is bounded on \([0,R]\) and zero elsewhere. For the second term we use that fact that for all \(x > R\),

    \[ x^{\alpha }e^{-x} = x^{M}e^{-x}.x^{\alpha - M} < \epsilon x^{\alpha - M},\]

    and which is thus integrable by Example 3.8.1. So the result follows by Lemma 3.6.3.

  • Remark 3.8.3 Let \(I\sw \R \) be an interval and recall \((I,\mc {B}(I),\lambda _I)\) from Remark 1.5.3, which are the restriction of the Borel sets and Lebesgue measure to the interval \(I\). A similar restriction applies for integrals, that is if \(f:\R \to \R \) then \(\int _\R \1_{I}(x)f(x)\,d\lambda (x)=\int _I f(x)\,d\lambda _I(x)\), or in more standard notation if \(I=[a,b]\) (for example) then

    \[\int _a^b f(x)\,dx=\int _\R \1_{[a,b]}(x)f(x)\,dx.\]

    We often treat this result as so obvious that we use it for free, and you may do so within this course. Weâ€™ll do so in the next example.

    Strictly, it does requires a proof, which can be done by first proving the result explicitly for simple functions, then applying the monotone convergence theorem to extend to non-negative measurable functions, and lastly transferring to integrable functions. As can be seen from the proof, there is nothing special about intervals here: in fact \(I\) can be be any Borel set.

  • Example 3.8.4 We want to calculate

    \[\lim _{n\to \infty } \int _0^1 \frac {nx^{2}}{nx + 5}\,dx.\]

    Weâ€™ll work in the measure space \(([0,1], {\cal B}([0, 1]), \lambda )\) and consider the sequence of functions \((f_{n})\) where \(f_{n}(x) = \frac {nx^{2}}{nx + 5}\) for all \(x \in [0,1], \nN \). Each \(f_{n}\) is continuous, hence measurable by Corollary 2.3.2. It is straightforward to check that \(\limn f_{n}(x) = x\) for all \(x \in [0,1]\) and that \(|f_{n}(x)| \leq 1\) for all \(\nN , x \in [0,1]\). So in this case, we can take \(K = 1\), and apply the Bounded Convergence Theorem to deduce that \(f(x) = x\) is integrable, and

    \[ \limn \int _{[0,1]}\frac {nx^{2}}{nx + 5}\,dx = \int _{[0,1]}x \,dx\]

    and we finish by evaluating \(\int _{[0,1]}x\, dx = \left [\frac {x^{2}}{2}\right ]_{0}^{1} = \frac {1}{2}.\)

  • Example 3.8.5 Summation of series is a special case of Lebesgue integration. Suppose that we are interested in \(\sum _{n=1}^{\infty }a_{n}\), where \(a_{n} \geq 0\) for all \(\nN \). We consider the sequence \((a_{n})\) as a function \(a:{\mathbb N} \rightarrow [0, \infty )\). We work with the measure space \(({\mathbb N}, {\mathcal P}({\mathbb N}), m)\) where \(m\) is counting measure. Then every sequence \((a_{n})\) gives rise to a nonâ€“negative measurable function \(a\) and

    \[ \sum _{n=1}^{\infty }a_{n} = \int _{{\mathbb N}}a(n)\,dm(n),\]

    which is for you to show in Problem 3.7. The same formula holds for general \((a_n)\sw \R \) provided that \(\sum _n |a_n|<\infty \) i.e. that \(a\) is integrable.

    The monotone and dominated converge theorems now provide tools for working with sequences of series. For example, suppose that for each \(m\in \N \) we have a sequence \((a_m(n))_{n\in \N }\), given by

    \[a_m(n)=\frac {1}{n^3}+\frac {1}{1+m^2n^2}.\]

    We canâ€™t easily compute the value of \(\sum _{n\in \N } a_m(n)\) for any given \(m\). But we can note that \(a_m(n)\to \frac {1}{n^3}\) as \(m\to \infty \), for all \(n\), and that \(|a_m(n)|\leq g(n)=\frac {1}{n^3}+\frac {1}{n^2}\). We know from analysis that \(\sum _n \frac {1}{n^3}\) and \(\sum _n \frac {1}{n^2}\) are both finite, so \(g\) is integrable â€“ which in this setting is just the claim that \(\sum _n g(n)<\infty \). So the Dominated Convergence Theorem applies, and we obtain

    \[\lim _{m\to \infty }\sum _{n\in \N }\l (\frac {1}{n^3}+\frac {1}{1+m^2n^2}\r )=\sum _{n\in \N }\frac {1}{n^3}.\]
