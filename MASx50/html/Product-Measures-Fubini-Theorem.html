Chapter 6 Product Measures and Fubini’s Theorem \((\Delta )\)

Note that this chapter is marked with a (\(\Delta \)). This means it is only included in MAS452/6352. It is not included in MAS350. It extends Sections 1.8 and 3.9.1.

In this chapter, we give brief notes which are intended as a summary of additional reading that you are expected to do outside the lectures. They emphasise the main ideas and concepts, but you will need to work carefully through all the relevant proofs. The recommended source is Adams and Guillemin, section 2.5, pp.89-102. This material is examinable. It can be studied straight after Chapter 3.

The aim of this section is to learn more about product measures, and how the theory of Lebesgue integration deals with multiple (double, triple, etc.) integrals. Before delving further into the details of these ideas, we’ll need an additional tool from the theory of \(\sigma \)-fields.

6.1 Dynkin’s \(\pi -\lambda \) Lemma \((\Delta )\)

Let \((S, \Sigma )\) be a measurable space. A collection \(\cal P\) of sets in \(S\) is called a \(\pi \)-system if \(A, B \in {\cal P} \Rightarrow A \cap B \in {\cal P}\) (i.e. \(\cal P\) is closed under intersections).

A collection \(\cal L\) of sets in \(S\) is called a \(\lambda \)-system if

  • (L1) \(S \in {\cal L}\).

  • (L2) If \((E_{n})\) is an increasing sequence of sets in \(\cal L\) (so \(E_{n} \subseteq E_{n+1}\) for all \(\nN \)), then \(\bigcup _{n=1}^{\infty }E_{n} \in {\cal L}\).

  • (L3) If \(E, F \in {\cal L}\) and \(F \subset E\) then \(E - F \in {\cal L}\).

Note that by (L1) and (L3), \(\lambda \)-systems are closed under complements.

  • Proposition 6.1.1 If \(\cal L\) is a \(\lambda \)-system that is also a \(\pi \)-system, then it is a \(\sigma \)-field.

Proof: Since \(\cal L\) is closed under complements and finite intersections, it is closed under finite unions by de Morgan’s laws. To show that \(\cal L\) is a \(\sigma \)-algeba, we need to prove that if \((A_{n})\) is an arbitrary sequence of sets in \(\cal L\), then \(\bigcup _{\nN }A_{n} \in {\cal L}\). This follows by writing \(\bigcup _{\nN }A_{n} = \bigcup _{\nN }B_{n}\) and using (L2), where

\[ B_{1} = A_{1}, B_{2} = B_{1} \cup (A_{2} - B_{1}), B_{3} = B_{2} \cup (A_{3} - B_{2}) \ldots \]

  ∎

Recall that if \(\cal A\) is a collections of sets in \(S\), then \(\sigma ({\cal A})\) is the smallest \(\sigma \)-field which contains \(\cal A\).

  • Lemma 6.1.2 (Dynkin’s \(\pi -\lambda \) Lemma) If \(\cal P\) is a \(\pi \)-system and \(\cal L\) is a \(\lambda \)-system with \({\cal P} \subseteq {\cal L}\), then \(\sigma ({\cal P}) \subseteq {\cal L}\).

Proof:   It suffices to prove that \({\cal L}({\cal P})\), which is the smallest \(\lambda \)-system that contains \(\cal P\), is a \(\sigma \)-field. By Proposition 6.1.1, its enough to prove that \({\cal L}({\cal P})\) is closed under intersections.

Step 1. Fix \(A \in {\cal L}({\cal P})\) and define

\[ {\cal G}_{A} = \{B \subseteq \Omega ; A \cap B \in {\cal L}({\cal P})\}.\]

You should check that \({\cal G}_{A}\) is a \(\lambda \)-system.

Step 2. If \(A, B \in {\cal P}\), then \(A \cap B \in {\cal P} \subseteq {\cal L}({\cal P})\). Hence \(B \in {\cal G}_{A}\). So we’ve shown that \({\cal P} \subseteq {\cal G}_{A}\), when \(A \in {\cal P}\). But in Step 1, we proved that \({\cal G}_{A}\) is a \(\lambda \)-system, and so since \({\cal L}({\cal P})\) is the smallest such, we deduce that \({\cal L}({\cal P}) \subseteq {\cal G}_{A}\). This means in particular that if \(A \in {\cal P}, B \in {\cal L}({\cal P})\), then \(A \cap B \in {\cal L}({\cal P})\).

Step 3. If \(A \in {\cal L}({\cal P})\), then by Step 2 we have \(A \cap B = B \cap A \in {\cal L}({\cal P})\) when \(B \in {\cal P}\). This shows that \({\cal P} \subseteq {\cal G}_{A}\), when \(A \in {\cal L}({\cal P})\). Now using Step 1 again, we find that \({\cal L}({\cal P}) \subseteq {\cal G}_{A}\), and the result then follows.   ∎

  • Corollary 6.1.3 If \(\mc {P}\) is a \(\pi \)-system and \(\mc {F}\) is a \(\sigma \)-field then \(\sigma (\mc {P})\sw \mc {F}\).

This result can be very useful for proving that a function is measurable.
