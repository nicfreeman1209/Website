2.2 Measurable Functions - Basic Concepts

In Section 1.6 we saw the existence subsets of \(\R \) that were not measurable, with respect to Lebesgue measure. These sets had no meaningful concept of ‘length’. We are looking to build a theory of integration in Chapter 3, but for such \(A\) integrating the indicator function \(\1_A\) would be dangerously close to trying to measure the length of \(A\), which we know we cannot do. Since \(\1_A\) is a function, we conclude that, just as with sets, we need a similar concept of a measurable function. In this section, we deal with functions \(f:S\to \R \), where \((S,\Sigma )\) is a measurable space.

We say that a function \(f:S\to \R \) is measurable if \(f^{-1}((a,\infty ))\in \Sigma \) for all \(a\in \R \). We will spend the rest of this section finding lots of different, and mathematically equivalent, ways of writing this same definition.

  • Theorem 2.2.1 Let \(f:S \rightarrow \R \). The following are equivalent:

    • (i) \(f^{-1}((a, \infty )) \in \Sigma \) for all \(a \in \R \) (i.e. \(f\) is measurable).

    • (ii) \(f^{-1}([a, \infty )) \in \Sigma \) for all \(a \in \R \).

    • (iii) \(f^{-1}((-\infty ,a)) \in \Sigma \) for all \(a \in \R \).

    • (iv) \(f^{-1}((-\infty , a]) \in \Sigma \) for all \(a \in \R \).

Proof: (i) \(\Leftrightarrow \) (iv) as \(f^{-1}(A)^{c} = f^{-1}(A^{c})\) and \(\Sigma \) is closed under taking complements.

(ii) \(\Leftrightarrow \) (iii) is proved similarly.

(i) \(\Rightarrow \) (ii) uses \([a, \infty ) = \bigcap _{n=1}^{\infty }(a-1/n, \infty )\) and so

\[ f^{-1}([a, \infty )) = \bigcap _{n=1}^{\infty }f^{-1}((a-1/n, \infty ))\]

and the result follows since \(\Sigma \) is closed under countable intersections.

(ii) \(\Rightarrow \) (i) uses

\[ f^{-1}((a, \infty )) = \bigcup _{n=1}^{\infty }f^{-1}([a+1/n, \infty ))\]

and the fact that \(\Sigma \) is closed under countable unions.   ∎

It follows that \(f\) is measurable if any of (i) to (iv) in Theorem 2.2.1 is established for all \(a \in \R \). In Problem 2.4 you can show that \(f\) is measurable if and only if \(f^{-1}((a,b)) \in \Sigma \) for all \(-\infty \leq a < b \leq \infty \).

A set \(O\) in \(\R \) is open if for every \(x \in O\) there is an open interval \(I\) containing \(x\) for which \(I \subseteq O\). It follows that every open interval in \(\R \) is an open set. We might ask what other kinds of open set there are. The following result gives a surprisingly clear answer, which implies that all open subsets of \(\R \) are Borel sets.

  • Proposition 2.2.2 Every open set \(O\) in \(\R \) is a countable union of disjoint open intervals.

Proof: Note that a ‘countable union’ includes the case where we only need finitely many intervals. Let us first note that if \(O_i\) are opens sets for all \(i\in I\) then (even if \(I\) is uncountable) the set \(O=\cup _i O_i\) is open. This is left as an exercise for you.

For \(x \in O\), let \(I_{x}\) be the union of all open intervals containing \(x\) for which \(I_x\sw O\). Then \(I_x\) is open. Also, \(I_x\) is an interval, because if \(a<b<c\) with \(a,b\in I_x\) then there are open intervals \((a-\epsilon _1,x+\epsilon _2)\) and \((x-\epsilon _3,c+\epsilon _4)\) within \(I_x\) and \(b\) is within their union, so \(b\in I_x\).

If \(x,y \in O\) and \(x \neq y\) then \(I_{x}\) and \(I_{y}\) are either disjoint or identical. To see this, note that if \(I_x\cap I_y\) is non-empty then \(I_x\cup I_y\) is a non-empty open interval contained within \(O\), which implies \(I_x\cup I_y\) is also contained within both \(I_x\) and \(I_y\). Thus \(I_x=I_y\).

However, there can only be countably many different \(I_x\), because we can only fit at most countably many (non-empty) disjoint open intervals within \(\R \). We now select a rational number \(r(x)\) in every distinct \(I_{x}\) and rewrite \(O\) as the countable disjoint union over intervals \(I_{x}\) labelled by distinct rationals \(r(x)\).   ∎

  • Theorem 2.2.3 The mapping \(f:S \rightarrow \R \) is measurable if and only if \(f^{-1}(O) \in \Sigma \) for all open sets \(O\) in \(\R \).

Proof: Suppose that \(f^{-1}(O) \in \Sigma \) for all open sets \(O\) in \(\R \). Then in particular \(f^{-1}((a, \infty )) \in \Sigma \) for all \(a \in \R \) and so \(f\) is measurable. Conversely assume that \(O\) is open in \(\R \) and use Proposition 2.2.2 to write \(O = \bigcup _{n=1}^{\infty }(a_{n}, b_{n})\). Then

\[ f^{-1}(O) = \bigcup _{n=1}^{\infty }f^{-1}((a_{n}, b_{n})).\]

If \(f\) is measurable, then \(f^{-1}((a_{n}, b_{n})) \in \Sigma \) for all \(\nN \) by Problem 2.4, and so \(f^{-1}(O) \in \Sigma \) since \(\Sigma \) is closed under countable unions.   ∎

  • Theorem 2.2.4 The mapping \(f:S \rightarrow \R \) is measurable if and only if \(f^{-1}(A) \in \Sigma \) for all \(A \in {\cal B}(\R )\).

Proof: Suppose that \(f\) is measurable and let \({\cal A} = \{E \subseteq \R ; f^{-1}(E) \in \Sigma \}\). We first show that \(\cal A\) is a \(\sigma \)-field.

(S1). \(\R \in {\cal A}\) as \(S = f^{-1}(\R )\).

(S2). If \(E \in {\cal A}\) then \(E^{c} \in {\cal A}\) since \(f^{-1}(E^{c}) = f^{-1}(E)^{c} \in \Sigma \).

(S3). If \((A_{n})\) is a sequence of sets in \(\cal A\) then \(\bigcup _{\nN }A_{n} \in {\cal A}\) since \(f^{-1}\left ( \bigcup _{n}A_{n}\right ) = \bigcup _{n}f^{-1}(A_{n}) \in \Sigma \).

By Problem 2.4, \(f^{-1}((a,b)) \in \Sigma \) for all \(-\infty \leq a < b \leq \infty \), and so \(\cal A\) is a \(\sigma \)-field of subsets of \(\R \) that contains all the open intervals. But by definition, \({\cal B}(\R )\) is the smallest such \(\sigma \)-field. It follows that \({\cal B}(\R ) \subseteq {\cal A}\) and so \(f^{-1}(A) \in \Sigma \) for all \(A \in {\cal B}(\R )\).

The converse is easy (e.g. just allow \(A\) to range over open sets, and use Theorem 2.2.3).   ∎

  • Remark 2.2.5 (\(\star \)) Based on Theorem 2.2.4, the idea of a measurable function can be extended beyond the real numbers. Let \((S_{1}, \Sigma _{1})\) and \((S_{2}, \Sigma _{2})\) be measurable spaces. The mapping \(f:S_{1} \rightarrow S_{2}\) is said to be measurable if \(f^{-1}(A) \in \Sigma _{1}\) for all \(A \in \Sigma _{2}\). We’ll stay within the real numbers in this course.
