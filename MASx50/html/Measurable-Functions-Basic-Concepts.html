2.2 Measurable Functions - Basic Concepts

We begin with some motivation from probability. Let \((\Omega , {\cal F}, \P )\) be a probability space. When we first study probability, we learn that random variables should be considered as mappings from \(\Omega \) to \(\R \). But is this enough for a rigorous mathematical theory? In practise we are interesting in calculating probabilities such as Prob\((X > a)\) where \(a \in \R \). What does this mean in terms of the measure \(\P \)? We must have

\[\begin {aligned} \mbox {Prob}(X > a) & = \P (\{\omega \in \Omega ; X(\omega ) \in (a, \infty )\})\\ & = \P (X^{-1}((a, \infty ))). \end {aligned}\]

Now \(X^{-1}((a, \infty )) \subseteq \Omega \), however \(\P \) only makes sense when applied to sets in \(\cal F\). So we conclude that \(\P (X > a)\) only makes sense if we impose an additional condition on the mapping \(X\), namely that \(X^{-1}((a, \infty )) \in {\cal F}\) for all \(a \in \R \). This property is precisely what we mean by measurability.

In fact let \((S, \Sigma )\) be an arbitrary measurable space. A mapping \(f:S \rightarrow \R \) is said to be measurable if \(f^{-1}((a, \infty )) \in \Sigma \) for all \(a \in \R \). So in particular, we define a random variable on a probability space to be a measurable mapping from \(\Omega \) to \(\R \).

  • Theorem 2.2.1 Let \(f:S \rightarrow \R \) be a mapping. The following are equivalent:

    • (i) \(f^{-1}((a, \infty )) \in \Sigma \) for all \(a \in \R \).

    • (ii) \(f^{-1}([a, \infty )) \in \Sigma \) for all \(a \in \R \).

    • (iii) \(f^{-1}((-\infty ,a)) \in \Sigma \) for all \(a \in \R \).

    • (iv) \(f^{-1}((-\infty , a]) \in \Sigma \) for all \(a \in \R \).

Proof: (i) \(\Leftrightarrow \) (iv) as \(f^{-1}(A)^{c} = f^{-1}(A^{c})\) and \(\Sigma \) is closed under taking complements.

(ii) \(\Leftrightarrow \) (iii) is proved similarly.

(i) \(\Rightarrow \) (ii) uses \([a, \infty ) = \bigcap _{n=1}^{\infty }(a-1/n, \infty )\) and so

\[ f^{-1}([a, \infty )) = \bigcap _{n=1}^{\infty }f^{-1}((a-1/n, \infty ))\]

and the result follows since \(\Sigma \) is closed under countable intersections.

(ii) \(\Rightarrow \) (i) uses

\[ f^{-1}((a, \infty )) = \bigcup _{n=1}^{\infty }f^{-1}([a+1/n, \infty ))\]

and the fact that \(\Sigma \) is closed under countable unions.   ∎

It follows that \(f\) is measurable if any of (i) to (iv) in Theorem 2.2.1 is established for all \(a \in \R \). In Problem 2.4 you can show that \(f\) is measurable if and only if \(f^{-1}((a,b)) \in \Sigma \) for all \(-\infty \leq a < b \leq \infty \).

A set \(O\) in \(\R \) is open if for every \(x \in O\) there is an open interval \(I\) containing \(x\) for which \(I \subseteq O\). It follows that every open interval in \(\R \) is an open set. We might ask what other kinds of open set there are. The following result gives a surprisingly clear answer.

  • Proposition 2.2.2 Every open set \(O\) in \(\R \) is a countable union of disjoint open intervals.

Proof: Note that a â€˜countable unionâ€™ includes the case where we only need finitely many intervals.

For \(x \in O\), let \(I_{x}\) be the largest open interval containing \(x\) for which \(I_{x} \subseteq O\). If \(x,y \in O\) and \(x \neq y\) then either \(I_{x}\) and \(I_{y}\) are disjoint or identical, for if they have a non-empty intersection their union is an open interval containing both \(x\) and \(y\) and that leads to a contradiction unless they coincide. Clearly \(O = \bigcup _{x \in O}I_{x}\). We now select a rational number \(r(x)\) in every interval \(I_{x}\) and rewrite \(O\) as the countable disjoint union over intervals \(I_{x}\) labelled by distinct rationals \(r(x)\).   ∎

From Proposition 2.2.2 we see that if \(O\) is an open set in \(\R \) then \(O \in {\cal B}(\R )\).

  • Theorem 2.2.3 The mapping \(f:S \rightarrow \R \) is measurable if and only if \(f^{-1}(O) \in \Sigma \) for all open sets \(O\) in \(\R \).

Proof: Suppose that \(f^{-1}(O) \in \Sigma \) for all open sets \(O\) in \(\R \). Then in particular \(f^{-1}((a, \infty )) \in \Sigma \) for all \(a \in \R \) and so \(f\) is measurable. Conversely assume that \(O\) is open in \(\R \) and use Proposition 2.2.2 to write \(O = \bigcup _{n=1}^{\infty }(a_{n}, b_{n})\). Then

\[ f^{-1}(O) = \bigcup _{n=1}^{\infty }f^{-1}((a_{n}, b_{n})).\]

If \(f\) is measurable, then \(f^{-1}((a_{n}, b_{n})) \in \Sigma \) for all \(\nN \) by Problem 2.4, and so \(f^{-1}(O) \in \Sigma \) since \(\Sigma \) is closed under countable unions.   ∎

We now present an equivalent, but more general result than Theorem 2.2.3.

  • Theorem 2.2.4 The mapping \(f:S \rightarrow \R \) is measurable if and only if \(f^{-1}(A) \in \Sigma \) for all \(A \in {\cal B}(\R )\).

Proof: Suppose that \(f\) is measurable and let \({\cal A} = \{E \subseteq \R ; f^{-1}(E) \in \Sigma \}\). We first show that \(\cal A\) is a \(\sigma \)-field.

S(i). \(\R \in {\cal A}\) as \(S = f^{-1}(\R )\).

S(ii). If \(E \in {\cal A}\) then \(E^{c} \in {\cal A}\) since \(f^{-1}(E^{c}) = f^{-1}(E)^{c} \in \Sigma \).

S(iii). If \((A_{n})\) is a sequence of sets in \(\cal A\) then \(\bigcup _{\nN }A_{n} \in {\cal A}\) since \(f^{-1}\left ( \bigcup _{\nN }A_{n}\right ) = \bigcup _{\nN }f^{-1}(A_{n}) \in \Sigma \).

By Problem 2.4, \(f^{-1}((a,b)) \in \Sigma \) for all \(-\infty \leq a < b \leq \infty \), and so \(\cal A\) is a \(\sigma \)-field of subsets of \(\R \) that contains all the open intervals. But by definition, \({\cal B}(\R )\) is the smallest such \(\sigma \)-field. It follows that \({\cal B}(\R ) \subseteq {\cal A}\) and so \(f^{-1}(A) \in \Sigma \) for all \(A \in {\cal B}(\R )\).

The converse is easy (e.g. just allow \(A\) to range over open sets, and use Theorem 2.2.3).   ∎

Theorem 2.2.4 leads to the following important extension of the idea of a measurable function: Let \((S_{1}, \Sigma _{1})\) and \((S_{2}, \Sigma _{2})\) be measurable spaces. The mapping \(f:S_{1} \rightarrow S_{2}\) is measurable if \(f^{-1}(A) \in \Sigma _{1}\) for all \(A \in \Sigma _{2}\).

Let \((S, \Sigma , m)\) be a measure space and \(f:S \rightarrow \R \) be a measurable function. It is easy to see that the mapping \(m_{f} = m \circ f^{-1}\) is a measure on \((\R , {\cal B}(\R )\). Indeed \(m_{f}(\emptyset ) = 0\) is obvious and if \((A_{n})\) is a sequence of disjoint sets in \({\cal B}(\R )\) we have

\[ \begin {aligned} m_{f}\left (\bigcup _{n=1}^{\infty }A_{n}\right ) & = m\left (f^{-1}\left (\bigcup _{n=1}^{\infty }A_{n}\right )\right )\\ & = m\left (\bigcup _{n=1}^{\infty }f^{-1}(A_{n})\right )\\ & = \sum _{n=1}^{\infty }m(f^{-1}(A_{n})) = \sum _{n=1}^{\infty }m_{f}(A_{n}), \end {aligned} \]

where we use the fact that for \(m \neq n, f^{-1}(A_{n}) \cap f^{-1}(A_{m}) = f^{-1}(A_{n} \cap A_{m}) = f^{-1}(\emptyset ) = \emptyset .\)

The measure \(m_{f}\) is called the pushforward of \(m\) by \(f\). In the case of a probability space \((\Omega , {\cal F}, P)\) and a random variable \(X: \Omega \rightarrow \R \), the pushforward is usually denoted \(p_{X}\). It is a probability measure on \((\R , {\cal B}(\R ))\) (you should check that it has total mass \(1\)) and is called the probability law or probability distribution of the random variable \(X\).
