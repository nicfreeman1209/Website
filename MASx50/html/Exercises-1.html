1.9 Exercises

  • 1.1 Show that if \(S\) is a set containing \(n\) elements, then the power set \({\cal P}(S)\) contains \(2^{n}\) elements.

    Hint: How many subsets are there of size \(r\), for a fixed \(1 \leq r \leq n\)? The binomial theorem may also be of some use.

  • 1.2 Let \(\Sigma _{1}\) and \(\Sigma _{2}\) be \(\sigma \)-fields of subsets of a set \(S\). Define

    \[ \Sigma _{1} \cap \Sigma _{2} = \{A \subseteq S; A \in \Sigma _{1}~\mbox {and}~A\in \Sigma _{2}\}.\]

    Show that \(\Sigma _{1} \cap \Sigma _{2}\) is a \(\sigma \)-field. Define \(\Sigma _{1} \cup \Sigma _{2} = \{A \subseteq S; A \in \Sigma _{1}~\mbox {or}~A\in \Sigma _{2}\}\). Why is \(\Sigma _{1} \cup \Sigma _{2}\) not in general a \(\sigma \)-field?

  • 1.3 If \((S,\Sigma ,m)\) is a measure space, show that for all \(A,B \in \Sigma \)

    • (a) \(m(A \cup B) + m(A \cap B) = m(A) + m(B),\)

    • (b) \(m(A \cup B) \leq m(A) + m(B).\)

    Hence prove that if \(A_{1}, A_{2}, \ldots , A_{n} \in \Sigma \),

    \[ m\left (\bigcup _{i=1}^{n}A_{i}\right ) \leq \sum _{i=1}^{n}m(A_{i}).\]

  • 1.4

    • (a) If \(m\) is a measure on \((S, \Sigma )\) and \(k > 0\), show that \(km\) is also a measure on \((S, \Sigma )\) where for all \(A \in \Sigma \),

      \[ (km)(A) = km(A).\]

      Hence show that if \(m\) is a finite measure and \(m(S)\neq 0\), then \(\P \) is a probability measure where \(\P (A) = \frac {m(A)}{m(S)}\) for all \(A \in \Sigma \).

    • (b) Let \([a, b]\) be a finite closed interval in \(\R \). Write down a formula for the uniform distribution as a probability measure on \(([a,b], {\mathcal B}([a, b]))\), using the above considerations and Lebesgue measure.

      Hint: Recall that the uniform distribution has the property that subintervals of \([a, b]\) which have the same length, will have the same probability.

    • (c) If \(m\) and \(n\) are measures on \((S, \Sigma )\), deduce that \(m + n\) is a measure on \((S, \Sigma )\) where \((m + n)(A) = m(A) + n(A)\) for all \(A \in \Sigma \).

  • 1.5

    • (a) If \(m\) is a measure on \((S, \Sigma )\) and \(B \in \Sigma \) is fixed, show that \(m_{B}(A) = m(A \cap B)\) for \(A \in \Sigma \) defines another measure on \((S, \Sigma )\).

    • (b) If \(m\) is a finite measure and \(m(B) > 0\), deduce that \(\P _{B}\) is a probability measure where

      \[\P _{B}(A) = \frac {m_{B}(A)}{m(B)}.\]

      How does this relate to the notion of conditional probability?

  • 1.6 Let \(S = \{x_{1}, x_{2}, \ldots , x_{n}\}\) be a finite set and let \(c_{1}, c_{2}, \ldots , c_{n}\) be non-negative numbers. Deduce that \(m\) is a measure on \((S, {\cal P}(S))\) where \(m = \sum _{i=1}^{n}c_{i}\delta _{x_{i}}\). What condition should be imposed on \(\{c_{1}, c_{2}, \ldots , c_{n}\}\) for \(m\) to be a probability measure?

  • 1.7 Show that \({\cal B}(\R )\) contains all closed intervals \([a, b]\), where \(-\infty < a < b < \infty \).

  • 1.8

    • (a) Let \(m\) be a finite measure on \((S, \Sigma )\).

      • (i) Show that, for any \(A\in \Sigma \),

        \[ m(A^{c}) = m(S) - m(A).\]

      • (ii) Let \((A_n)_{n\in \N }\) be a decreasing sequence of sets in \(\Sigma \). Show that \(m(A_{n})\to m(\cap _j A_j)\) as \(n\to \infty \).

    • (b) Give an example of a (not finite!) measure \(m\), and sets \(A_n\) and \(A = \bigcap _{n=1}^{\infty }A_{n}\) such that \(m(A) \neq \lim _{n \rightarrow \infty }m(A_{n})\).

Challenge Questions

  • 1.9 Let \(S\) be a finite set and \(\Sigma \) be a \(\sigma \)-field on \(S\). Consider the set

    \(\seteqnumber{0}{1.}{9}\)

    \begin{equation} \tag {$\star $}\label {eq:atoms} \Pi =\{ A \in \Sigma \- \text {if $B\in \Sigma $ and $B\sw A$ then either $B=A$ or $B=\emptyset $}\}. \end{equation}

    • (a) Show that \(\Pi \) is a finite set.

    • (b) Using (a), let us enumerate the elements of \(\Pi \) as \(\Pi =\{\Pi _1, \Pi _2,\cdots , \Pi _k\}\), where each \(\Pi _i\) is distinct from the others.

      • (i) Show that \(\Pi _i \cap \Pi _j= \emptyset \) for \(i\neq j\). Hint: Could \(\Pi _i \cap \Pi _j\) be an element of \(\Pi \)?

      • (ii) Show that \(\cup _{i=1}^k \Pi _i =S\). Hint: If \(C=S\setminus \cup _{i=1}^k \Pi _i\) is non-empty, is \(C\in \Pi \)?

      • (iii) Let \(A\in \Sigma \). Show that

        \[A=\bigcup _{i\in I}\Pi _i\]

        where \(I=\{i=1,\ldots ,k\- A\cap \Pi _i\neq \emptyset \}\).

  • 1.10 Prove that both of the follow claims are false.

    • (a) The Cantor set \(C\) contains an open interval \((a,b)\sw C\), where \(a<b\).

    • (b) If a Borel set has non-zero Lebesgue measure then it contains an open interval.
