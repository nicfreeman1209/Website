5.4 Characteristic Functions and Weak Convergence (\(\star \))

In this section, we introduce two tools that we will need to prove the central limit theorem. They rely on Lebesgue integration in \(\C \), which we introduced in Remark 3.6.4 and studied in exercises 3.17-3.21. Note that this section and the next are marked with a (\(\star \)), meaning that they are off-syllabus but are included for interest.

5.4.1 Characteristic Functions (\(\star \))

Let \((S, \Sigma , m)\) be a measure space and \(f:S \rightarrow \C \) be a complex-valued function. Then we can write \(f = f_{1} + i f_{2}\) where \(f_{1}\) and \(f_{2}\) are real-valued functions. We say that \(f\) is measurable/integrable if both \(f_{1}\) and \(f_{2}\) are. Define \(|f|(x) = |f(x)| = \sqrt {f_{1}(x)^{2} + f_{2}(x)^{2}}\) for each \(x \in S\). It is not difficult to see that \(|f|\) is measurable, using e.g. Problem 2.4. In Problem 5.11 we show that \(f\) is integrable if and only if \(|f|\) is integrable. With this definition, the Dominated Convergence Theorem continues to hold for sequences of measurable functions from \(S\) to \(\C \). We’ll omit a proof of this fact; it consists of applying our existing version of the theorem to real and imaginary parts.

  • Definition 5.4.1 Let \(X\) be a random variable defined on a probability space \((\Omega , {\cal F}, \P )\). The characteristic function \(\phi _{X}:\R \rightarrow \C \) of \(X\) and is defined, for each \(u \in \R \), by

    \(\seteqnumber{0}{5.}{5}\)

    \begin{equation} \label {eq:char_func} \phi _{X}(u) = \E [e^{iuX}] = \int _{\R }e^{iuy}p_{X}(dy). \end{equation}

Note that \(y \rightarrow e^{iuy}\) is measurable since \(e^{iuy} = \cos (uy) + i\sin (uy)\) and integrability holds since \(|e^{iuy}| \leq 1\) for all \(y \in \R \) and in fact we have \(|\phi _{X}(u)| \leq 1\) for all \(u \in \R \). Characteristic functions are unique to random variables in the following sense.

  • Theorem 5.4.2 If \(X\) and \(Y\) are two random variables for which \(\phi _{X}(u)=\phi _{Y}(u)\) for all \(u \in \R \) then \(p_{X} = p_{Y}\).

  • Example 5.4.3 Suppose that \(X \sim N(\mu , \sigma ^{2})\), that is \(X\) has a normal distribution (sometimes known as a Gaussian) with mean \(\mu \) and variance \(\sigma ^{2}\). In Problem 5.12 you can show for yourself that in this case \(\phi _{X}(u) = \exp \l (i\mu u - \frac {1}{2}\sigma ^{2}u^{2}\r )\) for all \(u \in \R \).

Equation 5.6 states that the characteristic function of \(X\) is the Fourier transform of the law \(p_{X}\) of the random variable \(X\). In elementary probability theory courses we often meet the Laplace transform \(\E [e^{uX}]\) of \(X\), which is called the moment generating function of \(X\). The moment generating function has the disadvantage that it only exists when \(|u|\) is small enough, because the function \(y \mapsto e^{uy}\) may not be integrable. How small \(|u|\) is required to be depends on \(X\), but unfortunately there are random variables for which \(\E [e^{uX}]\) is undefined for all \(u\neq 0\). In such cases the moment generating function is useless. The characteristic function has the important advantage that it is always defined for all \(u\in \R \).

Here is one useful property of characteristic functions, which is another instance of the ‘independence means multiply’ philosophy that we developed in Section 4.4.

  • Theorem 5.4.4 If \(X\) and \(Y\) are independent random variables then for all \(u \in \R \),

    \[ \phi _{X+Y}(u) = \phi _{X}(u)\phi _{Y}(u).\]

Proof: We have \(\phi _{X+Y}(u) = \E \big [e^{iu(X+Y)}\big ] = \E \l [e^{iuX}e^{iuY}\r ] = \E \l [e^{iuX}\r ]\E \l [e^{iuY}\r ] = \phi _{X}(u)\phi _{Y}(u)\), where the key step follows by Theorem 4.4.2.   ∎

5.4.2 Weak Convergence (\(\star \))

A sequence \((\mu _{n})\) of measures on \(\R \) is said to converge weakly to a measure \(\mu \) if

\[ \lim _{n \rightarrow \infty } \int _{\R } f(x) \,d\mu _{n}(x) = \int _{\R } f(x) \,d\mu (x) \]

for all bounded continuous functions \(f\) defined on \(\R \). This is a generalization of the convergence in law that we defined in Section 5.2, and is also equivalent to pointwise convergence of characteristic functions. The following theorem makes the connections precise. We won’t give a proof in this course.

  • Theorem 5.4.5 Let \(X_n,X\) be random variables with laws \(p_{X_n}\) and \(p_X\) (respectively), and characteristic functions \(\phi _n\) and \(\phi \). The following statements are equivalent:

    • 1. \(X_n\stackrel {d}{\to } X\);

    • 2. for all \(u\in \R \) we have \(\phi _n(u)\to \phi (u)\);

    • 3. for all bounded measurable functions \(f:\R \to \R \) we have

      \[\int _\R f(x)\,dp_{X_n}(x)\to \int _\R f(x)\,dp_X(x).\]

We’ve seen that there are several different modes of convergence for random variables. The same is true for measures, although we won’t make a list of them in this course. For further reading on this topic you might like to investigate ‘convergence in total variation’ and ‘vague convergence’.

5.4.3 An inequality concerning characteristic functions \((\star )\)

We will now develop an important inequality that we will need to prove the central limit theorem. Let \(x \in \R \) and let \(R_{n}(x)\) be the remainder term of the series expansion at \(n \in \mathbb {N} \cup \{0\}\) in \(e^{ix}\), i.e.

\[ R_{n}(x) = e^{ix} - \ds \sum _{k=0}^{n}\frac {(ix)^{k}}{k!}.\]

Note that \(R_{0}(x) = e^{ix} - 1 = \left \{\begin {array}{c c}\int _{0}^{x}ie^{iy}dy &~\mbox {if}~x > 0\\ -\int _{x}^{0}ie^{iy}dy &~\mbox {if}~x < 0 \end {array}\right .\). From the last two identities, we have \(|R_{0}(x)| \leq \min \{|x|, 2\}.\) Then you should check that \(R_{n}(x) = \left \{\begin {array}{c c} \int _{0}^{x}iR_{n-1}(y)dy&~\mbox {if}~x > 0\\ -\int _{x}^{0}iR_{n-1}(y)dy &~\mbox {if}~x < 0 \end {array}\right . \). Finally using induction, we can deduce the useful inequality:

\[ |R_{n}(x)| \leq \min \left \{\frac {2|x|^{n}}{n!}, \frac {|x|^{n+1}}{(n+1)!}\right \}.\]

Now let \(X\) be a random variable with characteristic function \(\phi _{X}\) for which \(\E (|X|^{n}) < \infty \) for some given \(\nN \). Then integrating the last inequality yields for all \(y \in \R \)

\(\seteqnumber{0}{5.}{6}\)

\begin{equation} \label {remain} \left |\phi _{X}(y) - \sum _{k=0}^{n}\frac {(iy)^{k}\E (X^{k})}{k!}\right | \leq \E \left [\min \left \{\frac {2|yX|^{n}}{n!}, \frac {|yX|^{n+1}}{(n+1)!}\right \}\right ]. \end{equation}

Equation (5.7) allows us to approximate a characteristic function with a polynomial, and quantify how large the error involved is. In some regions of \(y\in \R \) the error may be small, but in others it may be very large.

When we prove the CLT we will want to apply equation (5.7) in the case \(n=2\) to a random variable that has \(\E (X) = 0\). In this case, writing \(\E (X^{2}) = \sigma ^{2}\) we deduce that for all \(u \in \R \)

\(\seteqnumber{0}{5.}{7}\)

\begin{equation} \label {remain2} \left |\phi _{X}(y) - 1 + \frac {1}{2}\sigma ^{2}y^{2}\right | \leq \theta (y), \end{equation}

where \(\theta (y) = \ds y^{2}\E \left [\min \left \{|X|^{2}, |y|\frac {|X|^{3}}{6}\right \}\right ].\) Note that

\[\min \left \{|X|^{2}, |y|\frac {|X|^{3}}{6}\right \} \leq |X|^{2}\]

which is integrable by assumption. Also we have

\[\lim _{y \rightarrow 0}\min \left \{|X|^{2}, |y|\frac {|X|^{3}}{6}\right \} = 0,\]

and so by the dominated convergence theorem we can deduce that

\(\seteqnumber{0}{5.}{8}\)

\begin{equation} \label {remain3} \ds \lim _{y \rightarrow 0}\frac {\theta (y)}{y^{2}} = 0. \end{equation}
