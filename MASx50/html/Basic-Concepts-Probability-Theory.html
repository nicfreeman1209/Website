4.2 Basic Concepts of Probability Theory

4.2.1 Probability as Measure

Let us review what we know so far. In this chapter we will work with general probability spaces of the form \((\Omega , {\cal F}, \P )\) where the probability measure \(\P \) is a finite measure on \((\Omega , {\cal F})\) having total mass \(1\). So

\[ \P (\Omega ) = 1~~~\mbox {and}~~~0 \leq \P (A) \leq 1~\mbox {for all}~A \in {\cal F}.\]

Intuitively, \(\P (A)\) is the probability that the event \(A \in {\cal F}\) takes place. Since \(A \cup A^{c} = \Omega \) and \(A \cap A^{c} = \emptyset \), by (M2) we have \(1 = \P (\Omega ) = \P (A \cup A^{c}) = \P (A) + \P (A^{c})\) so that

\[ \P (A^{c}) = 1- \P (A).\]

A random variable \(X\) is a measurable function from \((\Omega , {\cal F})\) to \((\R , {\cal B}(\R ))\). If \(A \in {\cal B}(\R )\), it is standard to use the notation \((X \in A)\) to denote the event \(X^{-1}(A) \in {\cal F}\). The law or distribution of \(X\) is the induced probability measure on \((\R , {\cal B}(\R ))\) given by \(p_{X}(B) = \P (X^{-1}(B))\) for \(B \in {\cal B}(\R )\). So

\[ p_{X}(B) = \P (X \in B) = \P (X^{-1}(B)) = \P (\{\omega \in \Omega ; X(\omega ) \in B\}).\]

The expectation of \(X\) is the Lebesgue integral:

\[ \E (X) = \int _{\Omega }X(\omega )d\P (\omega ) \]

which makes sense and yields a finite quantity if and only if \(X\) is (Lebesgue) integrable, i.e. \(\E (|X|) < \infty \). In this case, we write \(\mu _X = \E (X)\) and call it the mean of \(X\). Note that for all \(A \in {\cal F}\)

\[ \P (A) = \E ({\1}_{A}).\]

By the result of Problem 2.6, any Borel measurable function \(f\) from \(\R \) to \(\R \) enables us to construct a new random variable \(f(X)\) for which \(f(X)(\omega ) = f(X(\omega ))\) for all \(\omega \in \Omega \). For example we may take \(f(x) = x^{n}\) for all \(\nN \). Then the \(n\)th moment \(\E (X^{n})\) will exist and be finite if \(|X|^{n}\) is integrable. If \(X\) has a finite second moment then its variance \(\var (X) =\E ((X-\mu )^{2})\) always exists (see Problem 4.12). It is common to use the notation \(\sigma _X^{2} = \var (X)\). The standard deviation of \(X\) is \(\sigma _X = \sqrt {\mbox {Var}(X)}\). When it is clear which random variable we mean, we write simply \(\mu \) and \(\sigma \) in place of \(\mu _X,\sigma _X\).

Hereâ€™s some useful notation. If \(X\) and \(Y\) are random variables defined on the same probability space and \(A_{1}, A_{2} \in {\cal B}(\R )\) it is standard to write:

\[ \P (X \in A_{1}, Y \in A_{2}) = \P ((X \in A_{1}) \cap (Y \in A_{2})).\]

4.2.2 Continuity of Probabilities

Recall from Section 1.7 that a sequence of sets \((A_{n})\) with \(A_{n} \in {\cal F}\) for all \(\nN \) is increasing if \(A_{n} \subseteq A_{n+1}\) for all \(\nN \). Similarly, we say that a sequence \((B_{n})\) with \(B_{n} \in {\cal F}\) for all \(\nN \) is decreasing if \(B_{n} \supseteq B_{n+1}\) for all \(\nN \).

  • Theorem 4.2.1 We have:

    • 1. Suppose \((A_{n})\) is increasing and \(A=\bigcup _n A_n\). Then \(\P [A] = \lim _{n \rightarrow \infty }\P [A_{n}]\).

    • 2. Suppose \((B_{n})\) is decreasing and \(B=\bigcap _n B_n\). Then \(\P [B] = \lim _{n \rightarrow \infty }\P [B_{n}]\).

Proof: This is just Theorem 1.7.1 of Chapter 1 applied to probability measures.   ∎

The intuition here should be clear. The set \(A_n\) gets bigger as \(n\to \infty \) and, in doing so, gets ever closer to \(A\); the same is true of their probabilities. Similarly for \(B_n\), which gets smaller and closer to \(B\). This result is a probabilistic analogue of the well known fact that monotone increasing (resp. decreasing) sequences of real numbers converge.

4.2.3 The Cumulative Distribution Function

Let \(X:\Omega \rightarrow \R \) be a random variable. Its cumulative distribution function or cdf is the mapping \(F_{X}:\R \rightarrow [0, 1]\) defined for each \(x \in \R \) by

\[F_{X}(x) = \P (X \leq x) = p_{X}((-\infty , x]).\]

When \(X\) is understood we will just denote \(F_{X}\) by \(F\). The next result gathers together some useful properties of the cdf. Recall that if \(f:\R \rightarrow \R ,\) the left limit at \(x\) is \(\lim _{y \uparrow x}f(y) = \lim _{y \rightarrow x, y < x}f(y)\), and the right limit at \(x\) is \(\lim _{y \downarrow x}f(y) = \lim _{y \rightarrow x, y > x}f(y)\). In general, left and right limits may not exist, but they do at every point when the function \(f\) is monotonic increasing (or decreasing).

  • Theorem 4.2.2 Let \(X\) be a random variable having cdf \(F\).

    • 1. \(\P (X > x) = 1 - F(x)\),

    • 2. \(\P (x < X \leq y) = F(y) - F(x)\) for all \(x < y\).

    • 3. \(F\) is monotone increasing, i.e \(F(x) \leq F(y)\) for all \(x < y\),

    • 4. \(\P (X=x) = F(x) - \lim _{y \uparrow x}F(y)\),

    • 5. The mapping \(x \rightarrow F(x)\) is right continuous, i.e. \(F(x) = \lim _{y \downarrow x}F(y)\), for all \(x \in \R \),

    • 6. \(\lim _{x \rightarrow -\infty }F(x) = 0,~~ \lim _{x \rightarrow \infty }F(x) = 1.\)

Proof: (1), (2) and (3) are easy exercises, left for you to do. (6) is Problem 4.4.

(4) Let \((a_{n})\) be a sequence of positive numbers that decreases to zero. Let \(x \in \R \) be arbitrary and for each \(\nN \), define \(B_{n} = (x - a_{n} < X \leq x)\). Then \((B_{n})\) decreases to the event \((X = x)\) and using (2) and Theorem 4.2.1 (2),

\[\P (X = x) = \lim _{n \rightarrow \infty }\P (B_{n}) = F(x) - \lim _{n \rightarrow \infty }F(x - a_{n}),\]

and the result follows.

(5) Let \(x\) and \((a_{n})\) be as in (4) and for each \(\nN \) define \(A_{n} = (X > x + a_{n})\). The sets \((A_{n})\) are increasing to \((X > x)\) and using (1) and Theorem 4.2.1 (1) we find that

\[ 1 - F(x) = \lim _{n \rightarrow \infty } \P (A_{n}) = 1 - \lim _{n \rightarrow \infty }F(x + a_{n}),\]

and the result follows.   ∎

4.2.4 Discrete and Continuous Random Variables

You will probably recall that many useful random variables are found in two special cases. Formally, we say that a random variable \(X\) is a:

  • 1. continuous random variable if its cdf \(F_{X}\) is continuous at every point \(x \in \R \);

  • 2. discrete random variable if \(F_{X}\) has jump discontinuities at a countable set of points and is constant between these jumps.

Note that if \(F_X\) is continuous at \(x\) then \(\P (X=x) = 0\) by Theorem 4.2.2(4). In particular, this applies to all \(x\in \R \) for continuous random variables. Many random random variables are neither discrete nor continuous.

We now point out a technicality that is often forgotten in less rigorous courses: a continuous random variable does not need to have a probability density function! Strictly speaking, those that do have a special name; we say \(X\) is a

  • 3. absolutely continuous random variable if there exists an integrable function \(f_{X}:\R \rightarrow \R \) so that \(F_{X}(x) = \int _{-\infty }^{x}f_{X}(y)dy\) for all \(x \in \R \).

The function \(f_{X}\) is called the probability density function or pdf of \(X\). Clearly \(f_{X} \geq 0\) (a.e.) and by Theorem 4.2.2 (6) we have \(\int _{-\infty }^{\infty }f_{X}(y)dy = 1\).

  • Lemma 4.2.3 Every absolutely continuous random variable is a continuous random variable.

Proof: Note that

\[\int _{-\infty }^{x}f_{X}(y)\,dy=\int _\R \1_{(y\leq x)}f_X(y)\,dy.\]

We want to prove this is a continuous function of \(x\), for which weâ€™ll use the dominated convergence theorem and the definition of continuity in terms of sequences. Let \((x_n)\sw \R \) be any sequence such that \(x_n\to x\). Note that \(|\1_{(y\leq x_n)}f_X(y)|\leq |f_X(y)|\) with \(\int _\R |f_X(y)|\,dy=\P [X\in \R ]=1<\infty \). If \(y\neq x\) then we have \(\1_{(y\leq x_n)}\to \1_{(y\leq x)}\) which means that \(\1_{(y\leq x_n)}f_X(y)\to \1_{(y\leq x)}f_X(y)\) for almost all \(y\in \R \). Hence by the dominated convergence theorem we have

\[\int _{-\infty }^{x_n}f_{X}(y)\,dy\to \int _{-\infty }^{x}f_{X}(y)\,dy\]

as \(n\to \infty \), as required.   ∎

We have already seen the example of the Gaussian random variable that is absolutely continuous. Most useful continuous random variables are absolutely continuous; other examples that you may have encountered previously include the uniform, exponential, Student \(t\), gamma and beta distributions. Typical examples of discrete random variables are the binomial, geometric and Poisson distributions.

4.2.5 Independence

In this subsection we consider the meaning of independence for events, random variables and \(\sigma \)-fields.

A useful heuristic is: independence means multiply. We say that two events \(A_{1}, A_{2} \in {\cal F}\) are independent if

\[ \P (A_{1} \cap A_{2}) = \P (A_{1})\P (A_{2}).\]

We extend this by induction to \(n\) events. But for many applications, we want to discuss independence of infinitely many events, or to be precise a sequence \((A_{n})\) of events with \(A_{n} \in {\cal F}\) for all \(\nN \). The definition of independence is extended from the finite case by considering all finite subsets of the sequence.

Formally: we say that the events in the sequence \((A_{n})\) are independent if the finite set \(\{A_{i_{1}}, A_{i_{2}}, \ldots , A_{i_{m}}\}\) is independent for all finite subsets \(\{i_{1}, i_{2}, \ldots , i_{m}\}\) of the natural numbers, i.e.

\[\P (A_{i_{1}} \cap A_{i_{2}} \cap \cdots , A_{i_{m}}) = \P (A_{i_{1}})\P (A_{i_{2}}) \cdots \P (A_{i_{m}}).\]

We recall that two random variables \(X\) and \(Y\) are said to be independent if \(\P (X \in A, Y \in B) = \P (X \in A)\P (Y \in B)\), for all \(A, B \in {\cal B}(\R )\). In other words the events \((X \in A)\) and \((Y \in B)\) are independent for all \(A, B \in {\cal B}(\R )\). Again this is easily extended to finite collections of random variables. Now suppose we are given a sequence of random variables \((X_{n})\). We say that the \(X_{n}\)â€™s are independent if every finite subset \(X_{i_{1}}, X_{i_{2}}, \ldots , X_{i_{m}}\) of random variables is independent, i.e.

\[ \P (X_{i_{1}} \in A_{i_{1}}, X_{i_{2}} \in A_{i_{2}}, \ldots , X_{i_{m}} \in A_{i_{m}}) = \P (X_{i_{1}} \in A_{i_{1}})\P (X_{i_{2}} \in A_{i_{2}}) \cdots \P (X_{i_{m}} \in A_{i_{m}})\]

for all \(A_{i_{1}}, A_{i_{2}}, \ldots , A_{i_{m}} \in {\cal B}(\R )\) and for all finite \(\{i_{1}, i_{2}, \ldots , i_{m}\} \subset \N \).

In the case where there are two random variables, we may consider the random vector \(Z = (X,Y)\) as a measurable function from \((\Omega , {\cal F})\) to \((\R ^{2}, {\cal B}(\R ^{2}))\) where the Borel \(\sigma \)-field \({\cal B}(\R ^{2})\) is the smallest \(\sigma \)-field generated by all open intervals of the form \((a, b) \times (c,d)\). The law of \(Z\) is, as usual, \(p_{Z} = \P \circ Z^{-1}\) and the joint law of \(X\) and \(Y\) is precisely \(p_{Z}(A \times B) = \P (X \in A, Y \in B)\) for \(A, B \in {\cal B}(\R )\). Then \(X\) and \(Y\) are independent if and only if

\[ p_{Z}(A \times B) = p_{X}(A)p_{Y}(B),\]

i.e. the joint law factorises as the product of the marginals.

  • Theorem 4.2.4 If \(X\) and \(Y\) are independent integrable random variables. Then

    \[ \E (XY) = \E (X)\E (Y).\]

Proof: (\(\star \)) We have

\[\begin {aligned} \E (XY) & = \int _{\R ^{2}}xyp_{Z}(dx, dy)\\ & = \left (\int _{\R }xp_{X}(dx)\right )\left (\int _{\R }yp_{Y}(dy)\right )\\ & = \E (X)\E (Y). \end {aligned}\]

Here, the first line is the two-dimensional version of Problem 4.10, and we have used Fubiniâ€™s theorem in the second line to write the integral over \(\R ^{2}\) as a repeated integral.   ∎

Lets go back to measure theory and consider a measurable space \((S, \Sigma )\). We say that \(\Sigma ^{\prime } \subseteq \Sigma \) is a sub-\(\sigma \)-field if it is itself a \(\sigma \)-field. For example the trivial \(\sigma \)-field \(\{S, \emptyset \}\) is a sub-\(\sigma \)-field of any \(\sigma \)-field defined on \(S\). If \((A_{n})\) is a sequence of sets in \(\Sigma \) then \(\sigma (A_{1}, A_{2}, \ldots )\) is defined to be the smallest sub-\(\sigma \)-field of \(\Sigma \) that contains \(A_{n}\) for all \(\nN \).

Sub-\(\sigma \)-fields play an important role in probability theory. For example let \(X\) be a random variable defined on \((\Omega , {\cal F}, \P )\). Then \(\sigma (X)\) is the smallest sub-\(\sigma \)-field of \(\cal F\) that contains all the events \(X^{-1}(A)\) for \(A \in {\cal B}(\R )\). For example let \(X\) describe a simple coin toss so that

\[X = \left \{\begin {array}{c c} 0 & \mbox {if the coin shows tails}\\ 1 & \mbox {if the coin shows heads}\end {array}\right .\]

If \(A = X^{-1}(\{0\})\) then \(A^{c} = X^{-1}(\{1\})\) and \(\sigma (X) = \{\emptyset , A, A^{c}, \Omega \}\).

Two sub-\(\sigma \)-fields \({\cal G}_{1}\) and \({\cal G}_{2}\) are said to be independent if

\[\P (A \cap B) = \P (A)\P (B)\]

for all \(A \in {\cal G}_{1}, B \in {\cal G}_{2}\). In Section 4.4 we will need the following proposition which we state without proof.

  • Proposition 4.2.5 Let \((A_{n})\) and \((B_{n})\) be sequences of events in \(\cal F\) which are such that the combined sequence \((A_{n}, B_{n})\) is independent (i.e. any finite subset containing both \(A_{n}\)â€™s and \(B_{m}\)â€™s is independent.) Then the two sub-\(\sigma \)-fields \(\sigma (A_{1}, A_{2}, \ldots )\) and \(\sigma (B_{1}, B_{2}, \ldots )\) are independent.

This result is very natural! It hopefully not hard to believe that it is true. A proof can be found in e.g. Rosenthal pp.28â€“29, but be aware that this requires the application of a rather deep result about \(\sigma \)-fields that we have also not proved in this course.
