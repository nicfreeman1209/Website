5.6 Exercises

  • 5.1 Let \(k\in \N \). Prove that in a sequence of independent coin tosses, infinitely many runs of \(k\) consecutive heads will occur.

  • 5.2 Let \((\Omega , \mc {F}, \P )\) be a probability space. Let \((A_n)\) be a sequence of events.

    • (a) Show that \(\{ A_{n}\text { e.v.}\} \subseteq \{A_{n}\text { i.o.}\}\).

    • (b) Show that \(\{A_n\text { i.o.}\}^{c} = \{A_n^c\text { e.v.}\}\) and deduce that \(\P [A_n\text { i.o.}] = 1 - \P [A_n^c\text { e.v.}].\)

    • (c) Show that

      \[\P [A_n\text { e.v.}]\;\leq \; \li \P [A_n]\;\leq \; \ls \P [A_{n}] \;\leq \; \P \l [A_{n}\text { i.o.}\r ].\]

  • 5.3 Let \(X\) be a real-valued random variable defined on a probability space \((\Omega , {\cal F}, P)\). Use the result of question 3.5 to derive the “probabilist’s” version of Chebychev’s inequality:

    \[ \P \l [|X - \mu | \geq c\r ] \leq \ds \frac {\var (X)}{c^{2}},\]

    where \(\var (X)\) and \(\E (X) = \mu \) are both assumed to be finite.

  • 5.4

    • (a) Suppose that \(X\) and \(Y\) are random variables and both \(X^{2}\) and \(Y^{2}\) are integrable. Prove the Cauchy-Schwarz inequality:

      \[ |\E [XY]| \leq (\E [X^{2}]^{\frac {1}{2}})(\E [Y^{2}]^{\frac {1}{2}}).\]

      Hint: Consider \(g(t) = \E [(X + tY)^{2}]\) as a quadratic function of \(t \in \R \).

    • (b) Deduce that if \(X^{2}\) is integrable then so is \(X\), and in fact \(|\E [X]|^{2} \leq \E [X^{2}]\).

    • (c) Let \(X\) be any random variable with a finite mean \(\E [X]=\mu \). Show that \(\E [X^{2}] < \infty \) if and only if \(\var (X) < \infty \).

  • 5.5 Let \(1\leq p\leq q\). Let \(X\) be a random variable for which \(\E [[X|^{p}] < \infty \). Show that

    \[\E [|X|^p]\leq 1+\E [|X|^q]\]

    and hence deduce that \(\E [|X|^q] < \infty \).

    Hint: Write \(X = X{\1}_{\{X \leq 1\}} + X{\1}_{\{X > 1\}}.\)

  • 5.6 Show that the following sequence \((X_n)\) and candidate limit \(X\) of random variables converges in probability but not almost surely. Take \(\Omega = [0,1],{\cal F} = {\cal B}([0,1])\) and \(\P \) to be Lebesgue measure. Take \(X = 0\) and define \(X_{n} = {\1}_{A_{n}}\) where \(A_{1} = [0, 1/2], A_{2} = [1/2, 1], A_{3} = [0, 1/4], A_{4} = [1/4, 1/2], A_{5} = [1/2, 3/4], A_{6} = [3/4, 1], A_{7} = [0, 1/8], A_{8} = [1/8, 1/4]\) etc.

  • 5.7 Show that the conclusion of the weak law of large numbers continue to hold if the requirement that the random variables \((X_{n})\) are i.i.d. is replaced by the weaker condition that they are identically distributed, and uncorrelated, i.e. \(\E (X_{m}X_{n}) = \E (X_{m})\E (X_{n})\) whenever \(m \neq n\).

    For this question, you should find an argument that doesn’t rely on the strong law!

  • 5.8

    • (a) Let \(X\) be a non-negative random variable and \(a > 0\). Show that \(\E (e^{-aX} ) \leq 1\).

    • (b) A random variable is said to have an exponential moment if \(\E (e^{a|X}) < \infty \) for some \(a > 0\). Show that \(X \sim N(0,1)\) has exponential moments for all \(a > 0\).

    • (c) If \(X\) has an exponential moment, deduce that it has moments to all orders, i.e. that \(\E (|X|^{n}) < \infty \) for all \(\nN \).

  • 5.9 (\(\star \)) Let \((S, \Sigma )\) be a measurable space and \(f: S \rightarrow \C \) be a (complex-valued) measurable function. Deduce that \(f\) is integrable if and only if \(|f|\) is.

  • 5.10 (\(\star \)) If \(X\) is a normally distributed random variable with mean \(\mu \) and variance \(\sigma ^{2}\), deduce that its characteristic \(\phi _{X}\) function is given for each \(u \in \R \) by \(\phi _{X}(u) = \exp {\left \{i\mu u - \frac {1}{2}\sigma ^{2}u^{2}\right \}}\).

    Hint: First show that it is sufficient to establish the case \(\mu = 0\) and \(\sigma = 1\) by writing \(Y = \frac {1}{\sigma }(X - \mu )\). Then show that \(y \rightarrow \phi _{Y}(u)\) is differentiable and deduce that \(\phi _{Y}^{\prime }(u) = -u\phi _{Y}(u)\). Now solve the initial value problem using what you know about \(\phi _{Y}(0)\).

  • 5.11 (\(\star \)) Suppose that \(X\) is a random variable for which \(\E (|X|^{n}) < \infty \) for some \(n\). Explain carefully why

    \[\E (X^{n}) = i^{-n}\left .\frac {d^{n}}{du^{n}}\phi _{X}(u)\right |_{u=0}.\]

  • 5.12 (\(\star \)) The first central limit theorem (CLT) to be established was due to de Moivre and Laplace. In this case each \(X_{n}\) takes only two values, \(1\) with probability \(p\) and \(0\) with probability \(1-p\) where \(0 < p < 1\) (i.e. the \(X_{n}\)s are i.i.d. Bernoulli random variables.) Write down the form of the CLT in this case (writing the standardised random variable \(Y_{n}\) in terms of \(S_{n} = X_{1} + X_{2} + \cdots + X_{n}\)), and explain its relation to the “binomial approximation to the normal distribution”.
