2.4 Algebra of Measurable Functions

One of our goals in this (and the next) section is to show that, sums, products, limits etc of measurable functions are themselves measurable. Throughout this section, \((S,\Sigma )\) is a measurable space.

Let \(f\) and \(g\) be functions from \(S\) to \(\R \) and define for all \(x \in S\),

\[ (f \vee g)(x) = \max \{f(x), g(x)\}~~~,~~~(f \wedge g)(x) = \min \{f(x), g(x)\}.\]

  • Proposition 2.4.1 If \(f\) and \(g\) are measurable then so are \(f \vee g\) and \(f \wedge g\).

Proof: This follows immediately from the facts that for all \(c \in \R \),

\[ (f \vee g)^{-1}((c, \infty )) = f^{-1}((c, \infty )) \cup g^{-1}((c, \infty ))\]

\[~~\mbox {and}~~(f \wedge g)^{-1}((c, \infty )) = f^{-1}((c, \infty )) \cap g^{-1}((c, \infty ))\]

  ∎

Let \(-f\) be the function \((-f)(x) = -f(x)\) for all \(x \in S\). If \(f\) is measurable it is easily checked that \(-f\) also is (take \(k=-1\) in Problem 2.5.)

Let \(\bf 0\) denote the zero function that maps every element of \(S\) to zero, i.e. \({\bf 0} = {\1}_{\emptyset }\). Then \(\bf 0\) is measurable since it is the indicator factor of a measurable set (or use Problem 2.3.)

Define \(f_{+} = f \vee {\bf 0}\) and \(f_{-} = -f \vee {\bf 0}\). So that for all \(x \in S\),

\[f_{+}(x) = \left \{\begin {array}{c c} f(x)& ~\mbox {if}~f(x) \geq 0\\ 0 & ~\mbox {if}~f(x) < 0 \end {array}\right ., f_{-}(x) = \left \{\begin {array}{c c} -f(x)& ~\mbox {if}~f(x) \leq 0\\ 0 & ~\mbox {if}~f(x) > 0 \end {array}\right .\]

  • Corollary 2.4.2 If \(f\) is measurable then so are \(f_{+}\) and \(f_{-}\).

Now define the set \(\{f > g\} = \{x \in S; f(x) > g(x)\}\).

  • Proposition 2.4.3 If \(f\) and \(g\) are measurable then \(\{f > g\} \in \Sigma \).

Proof:Let \(\{r_{n}, \nN \}\) be an enumeration of the rational numbers. Then

\[ \begin {aligned} \{f > g\} & = \bigcup _{\nN }\{f > r_{n} > g\}\\ & = \bigcup _{\nN }\{f > r_{n}\} \cap \{g < r_{n}\}\\ & = \bigcup _{\nN }f^{-1}((r_{n}, \infty )) \cap g^{-1}((-\infty , r_{n})) \in \Sigma \end {aligned} \]

  ∎

  • Theorem 2.4.4 If \(f\) and \(g\) are measurable then so is \(f + g\).

Proof: By Problem 2.5, we see that \(a-g\) is measurable for all \(a \in \R \). Now

\[(f + g)^{-1}((a, \infty )) = \{f + g > a\} = \{f > a - g\} \in \Sigma , \]

by Proposition 2.4.3 and this establishes the result.   ∎

You can use induction to show that if \(f_{1}, f_{2}, \ldots , f_{n}\) are measurable and \(c_{1}, c_{2}, \ldots , c_{n} \in \R \) then \(f\) is also measurable where \(f = c_{1}f_{1} + c_{2}f_{2} + \cdots + c_{n}f_{n}\). So the set of measurable functions from \(S\) to \(\R \) forms a real vector space. Of particular interest are the simple functions which take the form \(f = \sum _{i=1}^{n}c_{i}{\1}_{A_{i}}\) where \(A_{i} \in \Sigma ~( 1 \leq i \leq n)\). We will learn more about this highly useful class of functions in the next section and in Chapter 3 we will see that they play an important role in integration theory.

  • Theorem 2.4.5 If \(f:S \rightarrow \R \) is measurable and \(G:\R \rightarrow \R \) is continuous then \(G \circ f\) is measurable from \(S\) to \(\R \).

Proof: For all \(a \in \R \) let \(O_{a} = G^{-1}((a, \infty ))\). Then since \(G\) is continuous, \(O_{a}\) is an open set in \(\R \). Then since for any subset \(A\) of \(S\), \((G \circ f)^{-1}(A) = f^{-1}(G^{-1}(A))\), we have

\[ (G \circ f)^{-1}((a, \infty )) = f^{-1}(G^{-1}((a, \infty ))) = f^{-1}(O_{a}) \in \Sigma ,\]

by Theorem 2.2.2. The result follows.   ∎

  • Theorem 2.4.6 If \(f\) and \(g\) are measurable then so is \(fg\).

Proof: Apply Theorem 2.4.5 with \(G(x) = x^{2}\) to deduce that \(h^{2}\) is measurable whenever \(h\) is. But

\[ fg = \frac {1}{4}[(f + g)^{2} - (f-g)^{2}]\]

and the result follows by using Theorem 2.4.4.   ∎

Limits of Measurable Functions

Let \((f_{n})\) be a bounded sequence of functions from \(S\) to \(\R \) such that we also have the condition \(\sup _{\nN }\sup _{x \in S}|f_{n}(x)| < \infty \). Define \(\inf _{\nN }f_{n}\) and \(\sup _{\nN }f_{n}\) by

\[ \left (\inf _{\nN }f_{n}\right )(x) = \inf _{\nN }f_{n}(x)~~~\mbox {and}~~~\left (\sup _{\nN }f_{n}\right )(x) = \sup _{\nN }f_{n}(x)\]

for all \(x \in S\).1

  • Proposition 2.4.7 If \(f_{n}\) is measurable for all \(\nN \) then \(\inf _{\nN }f_{n}\) and \(\sup _{\nN }f_{n}\) are both measurable.

Proof: For all \(c \in \R \),

\[ \left \{\inf _{\nN }f_{n} > c\right \} = \bigcap _{\nN }\{f_{n} > c\} \in \Sigma .\]

\[ \left \{\sup _{\nN }f_{n} > c\right \} = \bigcup _{\nN }\{f_{n} > c\} \in \Sigma .\]

  ∎

We define \(\li f_{n}\) and \(\ls f_{n}\) by

\[ \left (\li f_{n}\right )(x) = \li f_{n}(x)~~~\mbox {and}~~~\left (\ls f_{n}\right )(x) = \ls f_{n}(x)\]

for all \(x \in S\)

  • Theorem 2.4.8 If \(f_{n}\) is measurable for all \(\nN \) then \(\li f_{n}\) and \(\ls f_{n}\) are both measurable.

Proof: By Proposition 2.4.7, \(\inf _{k \geq n}f_{k}\) and \(\sup _{k \geq n}f_{k}\) are measurable for each \(\nN \). Then by Proposition 2.4.7 again, \(\li f_{n} = \sup _{\nN }\inf _{k \geq n}f_{k}\) and \(\ls f_{n} = \inf _{\nN }\sup _{k \geq n}f_{k} \) are measurable.   ∎

We say that the sequence \((f_{n})\) converges pointwise to \(f\) as \(n \rightarrow \infty \) if \(\lim _{n \rightarrow \infty } f_{n}(x) = f(x)\) for all \(x \in S\).

  • Theorem 2.4.9 If \(f_{n}\) is measurable for all \(\nN \) and \((f_{n})\) converges pointwise to \(f\) as \(n \rightarrow \infty \), then \(f\) is measurable.

Proof: By Theorem 2.1.1 \(f(x) = \li f_{n}(x)\) for all \(x \in S\) and so \(f\) is measurable by Theorem 2.4.8.   ∎

1 We can drop the boundedness requirement if we work with functions taking values in \([-\infty , \infty ]\). See Section 2.6.
