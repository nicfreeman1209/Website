Chapter 2 Measurable Functions

We now restrict ourselves to studying a particular kind of function, known as a measurable function. For measure theory, this is an important step, because it allows us to exclude some very strangely behaved examples (in the style of Section 1.6) that would disrupt our theory.

We will see in Chapter 4 that measurable functions also play a huge role in probability theory, where they provide a mechanism for identifying which random variables depend on which information.

2.1 Liminf and Limsup

This section introduces some important tools from analysis which there wasnâ€™t time to cover in MAS221. Let \((a_{n})\) be a sequence of real numbers. It may or may not converge. For example the sequence whose \(n\)th term is \((-1)^{n}\) fails to converge but it does have two convergent subsequences corresponding to \(a_{2n-1} = -1\) and \(a_{2n} = 1\). This is a very special case of a general phenomenon that weâ€™ll now describe.

Assume that the sequence \((a_{n})\) is bounded, i.e. there exists \(K > 0\) so that \(|a_{n}| \leq K\) for all \(\nN \). Define a new sequence \((b_{n})\) by \(b_{n} = \inf _{k \geq n}a_{k}\). Then you can check that \((b_{n})\) is monotonic increasing and bounded above (by \(K\)). Hence it converges to a limit.

In fact, we are fine if \((a_n)\) is not bounded. In this case \(b_n\) is still monotone increasing, but we now have the extra possibilities that \(b_n\to \infty \) or \(b_n\to -\infty \). In fact, we might even have \(b_n=\pm \infty \) for some or all finite \(n\).

We define \(\li a_{n} = \lim _{n \rightarrow \infty }b_{n}\), where \(b_n=\inf _{k\geq n}a_k\). We call this the limit inferior or liminf of the sequence \((a_{n})\). So we have

\(\seteqnumber{0}{2.}{0}\)

\begin{equation} \label {li} \li a_{n} = \lim _{n \rightarrow \infty } \inf _{k \geq n}a_{k} = \sup _{\nN }\inf _{k \geq n}a_{k}. \end{equation}

Similarly the sequence \((c_{n})\) where \(c_{n} = \sup _{k \geq n}a_{k}\) is monotonic decreasing and bounded below. So it also converges to a limit which we call the limit superior or limsup for short. We denote \(\ls a_{n} = \lim _{n \rightarrow \infty }c_{n}\). Then we have

\(\seteqnumber{0}{2.}{1}\)

\begin{equation} \label {prop:ls} \ls a_{n} = \lim _{n \rightarrow \infty } \sup _{k \geq n}a_{k} = \inf _{\nN }\sup _{k \geq n}a_{k}. \end{equation}

Clearly we have \(\li a_{n} \leq \ls a_{n}\). In Problem 2.2, you can investigate some other properties of \(\limsup \) and \(\liminf \). It can be shown that the smallest limit of any convergent subsequence of \((a_{n})\) is \(\li a_{n}\), and the largest limit is \(\ls a_{n}\). The next theorem is very useful:

  • Theorem 2.1.1 A bounded sequence of real numbers \((a_{n})\) converges to a limit if and only if \(\li a_{n} = \ls a_{n}\). In this case we have

    \[ \lim _{n \rightarrow \infty } a_{n} = \li a_{n} = \ls a_{n}.\]

Proof: If \((a_{n})\) converges to a limit, then all of its subsequences also converge to the same limit and it follows that \(\lim _{n \rightarrow \infty } a_{n} = \li a_{n} = \ls a_{n}\). Conversely suppose that we donâ€™t know that \((a_{n})\) converges but we do know that \(\li a_{n} = \ls a_{n}\). Then for all \(\nN \),

\[ 0 \leq a_{n} - \inf _{k \geq n}a_{k} \leq \sup _{k \geq n}a_{k} - \inf _{k \geq n}a_{k}.\]

But

\[\lim _{n \rightarrow \infty }\left (\sup _{k \geq n}a_{k} - \inf _{k \geq n}a_{k}\right ) = \ls a_{n} - \li a_{n} = 0\]

and so

\[\lim _{n \rightarrow \infty } \left (a_{n} - \inf _{k \geq n}a_{k}\right ) = 0\]

by the sandwich rule. But since

\[ a_{n} = \left (a_{n} - \inf _{k \geq n}a_{k}\right ) + \inf _{k \geq n}a_{k},\]

and \(\lim _{n \rightarrow \infty } \inf _{k \geq n}a_{k} = \li a_{n}\), we can use the algebra of limits to deduce that \((a_{n})\) converges to the common value of \(\li a_{n}\) and \(\ls a_{n}\).   ∎
